### Vibe Coding 核心理念

> **Vibe Coding = 规划驱动 + 上下文固定 + AI 结对执行**

### 重构架构

```
Orchestrator (协调器)
       │
       ├──▶ Memory Bank (记忆库)
       │
       ├──▶ Planner Agent (规划师)
       │
       ├──▶ 专业 Agent 团队
       │   ├── Technical Agent (技术分析)
       │   ├── Sentiment Agent (情绪分析)
       │   ├── Fundamental Agent (基本面)
       │   ├── Risk Agent (风险评估)
       │   └── News Agent (新闻整合)
       │
       └──▶ Synthesizer (综合汇总)
```

### 关键创新

1. **Planner Agent**：制定分析计划，确定执行顺序
2. **Memory Bank**：维护上下文，防止信息丢失
3. **Vibe Check**：每步输出进行感觉校验
4. **多 Agent 协同**：专业分工 + 并行执行
