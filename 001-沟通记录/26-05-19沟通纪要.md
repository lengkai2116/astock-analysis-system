## 005 - 功能需求讨论

**主题**：三项新功能需求评估  
**时间**：2026-05-19  
**参与方**：用户、AI助手  

**需求概述**：

### 一、股票选择策略阶梯
| 层级 | 名称 | 功能 |
|------|------|------|
| 第一层 | 风险剔除 | 基于达尔文筛选系统剔除高风险股票 |
| 第二层 | 目标锁定 | 根据机构持股、量价阶段、主力入场信号筛选 |
| 第三层 | 买卖点验证 | 基于策略+大模型分析锁定高概率股票 |

### 二、自选股监控体系
| 功能 | 描述 |
|------|------|
| 自选股概览 | 核心数据和状态简述 |
| 持股机构变动 | 监控机构持仓变化 |
| 量价监控 | 近期量价变化监控 |
| 策略分析建议 | 三个策略的分析对应建议 |
| K线展示 | 主要数据及K线展示 |

### 三、复盘机制
| 维度 | 描述 |
|------|------|
| 策略有效性 | 评估胜率、盈亏比 |
| 市场量价变化 | 分析市场环境影响 |
| 盈亏风控管理 | 评估风险控制效果 |
| 仓位管理 | 评估仓位配置合理性 |

**评估结果**：
| 需求 | 可行性 | 优先级 | 建议阶段 |
|------|--------|--------|----------|
| 策略阶梯 | ✅ 高 | P0 | 阶段三+四 |
| 自选股监控 | ✅ 高 | P0 | 阶段三+五 |
| 复盘机制 | ✅ 中 | P1 | 阶段四 |

**需确认事项**：
1. 第二层筛选的具体条件（机构持股比例、量价阶段定义等）
2. 第三层策略选择（从15个策略中选3个）

**方案文档**：[058-功能需求评估报告.md](file:///Users/kalence/Desktop/测试/002-方案存档/058-功能需求评估报告.md)
