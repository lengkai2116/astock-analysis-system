#### 2. 信号生成逻辑

**买入信号** (score >= 3):
- 价格低于平均成本10%以上 (+2)
- 价格低于筹码峰5%以上 (+2)
- 筹码高度集中 (>0.6) (+1)
- 低位获利率低(<0.3)且价格偏离大(<-15%) (+2)
- 近5日跌幅超过10% (+1)

**卖出信号** (score <= -2):
- 高位获利率高(>0.6)且价格上涨(>5%) (-2)

#### 3. 主力阶段识别

| 阶段 | 英文 | 判断条件 |
|------|------|----------|
| 建仓 | BUILDING | 价格低于成本10%以下 + 集中度>40% |
| 洗盘 | WASHING | 高度集中 + 波动性高 |
| 拉升 | RAISING | 价格高于成本10% + 获利率>70% |
| 出货 | SHIPPING | 价格高位 + 集中度下降 |
| 下跌支撑 | SUPPORT | 低位 + 低获利率 |
