## 四、前后端集成说明

### 4.1 API对应关系

| 前端方法 | 后端API | 说明 |
|---------|---------|------|
| `factorService.getFactors()` | `/api/factors` | 获取因子列表 |
| `factorService.getCombinations()` | `/api/factors/combinations` | 获取组合 |
| `factorService.createCombination()` | `POST /api/factors/combinations` | 创建组合 |
| `factorService.backtestCombination()` | `/api/factors/backtest` | 回测 |
| `factorService.getStrategies()` | `/api/factors/strategies` | 策略列表 |

### 4.2 状态管理

```javascript
// Store结构
{
  factors: { all: [], loading: false },
  factorCombinations: { all: [], current: null },
  strategies: { available: [], active: [] },
  backtestResult: { data: null, loading: false }
}
```
