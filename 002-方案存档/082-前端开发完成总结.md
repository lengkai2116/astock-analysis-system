## 三、功能模块详解

### 3.1 因子API服务层 (`factorService.js`)

**功能**：
- 获取因子列表
- 获取因子详情
- 计算单个/批量因子
- 因子组合CRUD
- 回测接口
- 策略接口

**API端点**：
```javascript
factorService.getFactors()
factorService.createCombination(data)
factorService.backtestCombination(data)
factorService.getStrategies()
```

### 3.2 Vuex Store扩展

**新增状态**：
```javascript
state.factors          // 因子列表
state.factorCombinations  // 组合列表
state.strategies        // 策略列表
state.backtestResult    // 回测结果
```

**新增actions**：
- `loadFactors()` - 加载因子
- `loadFactorCombinations()` - 加载组合
- `loadStrategies()` - 加载策略
- `createCombination()` - 创建组合
- `updateCombination()` - 更新组合
- `deleteCombination()` - 删除组合
- `runBacktest()` - 运行回测

### 3.3 因子选择组件 (`FactorSelector`)

**功能**：
- 按分类展示因子（Qlib、GTJA、学术等）
- 搜索因子
- 多选因子
- 配置因子权重
- 实时预览已选因子

**UI特性**：
- 三栏布局（分类/因子/已选）
- 卡片式因子展示
- 权重滑块调整
- 分类颜色标签

### 3.4 因子组合管理页面 (`factor-manager`)

**功能**：
1. **我的组合**Tab
   - 列表展示所有组合
   - 创建/编辑/删除组合
   - 设置默认组合/收藏
   - 运行回测

2. **策略筛选**Tab
   - 选择因子组合
   - 选择股票池
   - 配置筛选数量
   - 展示筛选结果
   - 导出功能

**UI特性**：
- 卡片式组合展示
- 弹窗式编辑
- 表格化筛选结果
- 进度条得分展示

### 3.5 自选监控页面增强

**新增功能**：
- 策略中心下拉菜单
  - 因子组合管理
  - 策略筛选
  - 回测系统

**跳转方法**：
```javascript
openFactorManager()   // 打开因子管理
runStrategyScreen()   // 运行策略筛选
goToBacktest()       // 跳转回测
```

### 3.6 回测页面增强

**新增功能**：
1. **因子组合选择**
   - 下拉选择已有组合
   - 支持清空

2. **增强绩效指标**
   - 索提诺比率
   - 波动率
   - Beta
   - Alpha
   - 信息比率
   - Benchmark对比
   - 超额收益

3. **回撤图表**
   - 实时回撤曲线
   - 最大回撤标注

4. **因子组合标签**
   - 显示当前使用的因子组合

  ↓

## 七、页面跳转流程

### 流程1：自选股 → 因子分析
```
自选监控页面
  ↓ 点击"策略中心" → "因子组合管理"
因子组合管理页面
  ↓ 创建/选择组合 → "回测"

  ↓ 查看结果 → "分析"

```

  ↓ 点击"分析"
指标IDE页面
