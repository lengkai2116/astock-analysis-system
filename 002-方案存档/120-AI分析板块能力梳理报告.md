                           │

                           │
┌─────────────────────────────────────────────────────────┐
│                    引擎层（Engine）                       │
│  ┌────────────────────────────────────────────────┐   │
│  │           策略框架 (Framework)                  │   │
│  │  - 缠论策略 (chanlun_strategy.py)              │   │
│  │  - 筹码策略 (chip_strategy.py)                 │   │

## 二、核心能力详解

### 2.1 多角色AI分析师协作

#### 2.1.1 五大分析师角色

| 角色 | 名称 | 专业领域 | 核心能力 |
|------|------|----------|----------|
| technical | 技术分析师 | K线形态、技术指标、成交量 | 技术信号识别 |
| fundamental | 基本面分析师 | 财务报表、行业地位、竞争优势 | 价值评估 |
| macro | 宏观策略师 | 宏观经济、政策导向、资金流向 | 趋势研判 |
| risk | 风险控制官 | 市场风险、信用风险、流动性风险 | 风险识别 |
| fund_manager | 资深基金经理 | 价值投资、风险管理 | 投资建议 |

#### 2.1.2 分析师接口

**文件**: [ai_analysis.py](file:///Users/kalence/Desktop/01-A股股票分析系统/backend/app/routes/ai_analysis.py)

```python
# 核心接口
@ai_analysis_bp.route('/api/v3/ai/analyze', methods=['POST'])  # 启动分析
@ai_analysis_bp.route('/api/v3/ai/analysts', methods=['GET'])   # 获取分析师列表
@ai_analysis_bp.route('/api/v3/ai/analyst-progress', methods=['GET'])  # 获取进度
@ai_analysis_bp.route('/api/v3/ai/analyst-result', methods=['GET'])    # 获取单个结果
@ai_analysis_bp.route('/api/v3/ai/final-report', methods=['GET'])      # 最终报告
```

#### 2.1.3 投研报告输出

- **总体评级**: 强烈推荐 / 推荐 / 中性 / 谨慎
- **评分**: 7.5-9.5分
- **目标价**: 预测价格区间
- **止损位**: 风险控制点位
- **仓位建议**: 10-40%
- **时间周期**: 1-3个月 / 3-6个月 / 6-12个月

#### 2.2.1 核心能力

**文件**: [ai_strategy_generator.py](file:///Users/kalence/Desktop/01-A股股票分析系统/backend/app/services/ai_strategy_generator.py)

| 功能 | 描述 | 状态 |
|------|------|------|
| 自然语言转指标 | 从描述生成量化指标 | ✅ |
| 参数提取 | 自动识别指标参数 | ✅ |
| 公式生成 | 构建计算公式 | ✅ |
| 信号逻辑 | 买卖信号定义 | ✅ |
| 代码模板 | 可执行策略代码 | ✅ |
| 回测解读 | AI分析回测结果 | ✅ |

#### 2.2.2 支持的指标类型

| 类型 | 关键词 | 公式模板 |
|------|--------|----------|
| 移动平均 | 均线、MA、SMA、EMA | `MA(close, {period})` |
| RSI | 相对强弱、超买、超卖 | `RSI(close, {period})` |
| MACD | 指数平滑、异同移动 | `MACD(close, {fast}, {slow}, {signal})` |
| 布林带 | Bollinger、波动带 | `BOLL(close, {period}, {std})` |
| KDJ | 随机指标、随机震荡 | `KDJ(close, high, low, {n}, {m1}, {m2})` |
| 成交量比 | VOL、量比 | `vol / vol.mean({window})` |
| 动量指标 | momentum、ROC | `(close - close.shift({period})) / close.shift({period})` |

### 2.4 三层策略筛选架构

#### 2.4.1 架构设计

```
第一层: 达尔文策略（风险过滤）
├─ 涨幅过滤（过大涨幅剔除）
├─ 跌幅过滤（过大跌幅剔除）
├─ ST股票过滤
└─ 成交量过滤

第二层: 筹码策略（主力识别）
├─ 筹码分布分析
├─ 集中度计算
├─ 获利比例分析
└─ 主力介入评分

第三层: 策略验证（多策略融合）
├─ 缠论策略分析
├─ 因子策略验证
├─ 自定义策略支持
└─ SignalFusion信号融合
```
