```python
def detect_rsi_divergence(closes: np.ndarray, rsi_values: np.ndarray, window: int = 20) -> Dict:
    """检测RSI顶/底背离"""
    # 检查价格是否创新高/低
    price_new_high = closes[-1] >= np.max(closes[-window:])
    price_new_low = closes[-1] <= np.min(closes[-window:])
    
    # 找到前一个极值
    prev_high_idx = np.argmax(closes[-window:-1])  # 前一次高点
    prev_low_idx = np.argmin(closes[-window:-1])    # 前一次低点
    
    current_rsi = rsi_values[-1]
    
    # 顶背离: 价格新高，RSI不新高
    if price_new_high:
        prev_rsi_at_high = rsi_values[-(window - prev_high_idx)]
        if current_rsi < prev_rsi_at_high:
            return {'type': 'top_divergence', 'strength': prev_rsi_at_high - current_rsi}
    
    # 底背离: 价格新低，RSI不新低
    if price_new_low:
        prev_rsi_at_low = rsi_values[-(window - prev_low_idx)]
        if current_rsi > prev_rsi_at_low:
            return {'type': 'bottom_divergence', 'strength': current_rsi - prev_rsi_at_low}
    
    return {'type': 'none'}
```

#### P1-3 筹码转移速率量化

**依据**: PDF第4章 §4.2.3

**当前状态**: `calculate_transfer_rate()` 用标准差变化，不符合PDF公式

**修正方案**:

```python
def calculate_transfer_rate_v2(chip_bins_history, lookback=20) -> float:
    """
    PDF公式: 转移速度 = |Δ低位筹码占比| / 20日
    
    结果分档:
      > 1.0%/日 → 加速行情，高警觉
      0.3~1.0%/日 → 正常行情
      < 0.3%/日 → 蓄势阶段
    """
    earliest = chip_bins_history[-lookback-1]
    latest = chip_bins_history[-1]
    
    def low_chip_ratio(chips):
        prices = sorted(c['price_bin'] for c in chips)
        low_threshold = prices[0] + (prices[-1] - prices[0]) * 0.3
        return sum(c['chip_ratio'] for c in chips if c['price_bin'] <= low_threshold)
    
    earliest_ratio = low_chip_ratio(earliest)
    latest_ratio = low_chip_ratio(latest)
    
    speed = abs(latest_ratio - earliest_ratio) / lookback
    return speed  # 日化转移率
```

#### P2-2 五档盘口不平衡（大单流向Proxy）

**QMT数据**: ✅ 五档买卖盘口实时可用

> **说明**: 虽然无Level2逐笔成交，五档盘口仍可提供有效的"盘口不平衡"替代信号

```python
def calc_order_imbalance(bid_volumes: List[float], ask_volumes: List[float]) -> float:
    """计算五档盘口不平衡因子"""
    total_bid = sum(bid_volumes)
    total_ask = sum(ask_volumes)
    if total_bid + total_ask == 0:
        return 0
    return (total_bid - total_ask) / (total_bid + total_ask)  # -1 ~ +1

def calc_weighted_imbalance(bid_prices: List[float], bid_volumes: List[float],
                            ask_prices: List[float], ask_volumes: List[float]) -> float:
    """加权盘口不平衡（考虑价格深度）"""
    # 越近的价位权重越高
    weights = [5, 4, 3, 2, 1]  # 买一~买五 / 卖一~卖五
    total_bid_weighted = sum(v * w for v, w in zip(bid_volumes, weights))
    total_ask_weighted = sum(v * w for v, w in zip(ask_volumes, weights))
    if total_bid_weighted + total_ask_weighted == 0:
        return 0
    return (total_bid_weighted - total_ask_weighted) / (total_bid_weighted + total_ask_weighted)
```

## 七、核心结论

### 7.1 QMT补齐后策略可行边界

- **所有PDF核心信号**均可在Level1数据范围内实现
- **无需Level2**即可完成95%的筹码分布策略逻辑

1. **入出场信号**（P0-1 + P0-2）：SSRP穿越 + RSI背离 → 买卖点的精度最具杠杆效应
2. **风险控制**（P0-4 + P0-5）：止盈 + 仓位管理 → 对最大回撤影响最大
3. **信号辅助验证**（P0-3 + P2-2）：换手率 + 盘口不平衡 → 信号置信度提升
