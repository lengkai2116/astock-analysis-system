## 发现汇总

### 已确认正常项

- 全部 11 个页面正常渲染，无白屏、无 JS 运行时错误
- 后端 API（K 线、选股、账户、自选、因子、AI 分析）均返回结构正确数据
- 主题切换持久化（localStorage）工作正常
- 前端在 API 500/网络断开/畸形数据时均不崩溃，有恰当的错误处理
- 首页加载性能良好（1.6s），页面切换流畅

### 备注（非阻塞）

1. **侧边栏折叠状态**：刷新后恢复为展开状态（Pinia 状态未持久化到 localStorage，属设计行为）
2. **快捷键 `?` 帮助**：当前实现监听 `?` 键而非 `Shift+/`，Playwright `keyboard.press('?')` 未触发（需 `keyboard.press('Shift+/')`）
3. **后端数据**：部分 API（如 AI 分析、选股）依赖于实际市场数据和 LLM 服务，当前 mock 模式不会产生完整分析结果，但不影响前端渲染检查
