# 观潮网站系统能力调研与对标分析报告

> 调研目标：https://guanchaotv.com/app/
> 调研方法：CSS 反向解析(527KB) + JS 主包反编译(1.08MB) + 6个分包深度分析(共1.5MB)
> 调研日期：2026-06-05

---

## 1. 概述

观潮（GuanChao）是一个 **AI 原生的专业交易分析工作台**，面向 A 股/二级市场个人投资者的付费交易分析工具。全部前端源码通过 Vite 构建，由 15 个 ES module 分包组成。

### 1.1 完整技术栈

| 层 | 观潮 | 本项目 |
|---|------|--------|
| 框架 | **React 18** | Vue 3 |
| 状态管理 | **Zustand**（6 个 Store） | Pinia |
| UI 原语 | **Radix UI**（headless） | Ant Design Vue 4.x |
| 动画 | **Framer Motion** | 无 |
| 图表 | **ECharts** + 自研 K 线渲染 | KlineCharts 9.x + ECharts 5.x |
| 代码编辑器 | **Monaco Editor** | CodeMirror 5.x |
| 主字体 | **Inter Variable**（可变字体） | Inter（静态）|
| 等宽字体 | JetBrains Mono Variable | JetBrains Mono |
| 构建 | Vite（15 分包预加载） | Vite（3 分包）|
| AI 模型 | **阿里 Qwen（通义千问）** | DeepSeek API + LM Studio |
| 数据源 | **Tushare Pro** | Tushare Pro |
| 国际化 | 仅中文 | vue-i18n 已引入 |
| PWA | 完整 PWA + manifest + Service Worker | 无 |
| 存储 | IndexedDB + localStorage | 仅 localStorage |

## 5. 观潮的 5 个核心设计亮点

### 5.1 半透明叠加色体系

观潮没有使用多层级纯色背景，全部通过白色在深色底上的不同透明度构建层级：

```css
--bg-hover:          rgba(255,255,255,0.04)  /* 最浅 */
--bg-overlay-default: rgba(255,255,255,0.06) /* 次级面板 */
--bg-overlay-strong:  rgba(255,255,255,0.09) /* 强调面板 */
--bg-active:          rgba(255,255,255,0.08) /* 激活态 */
--bg-selected:        rgba(91,94,244,0.1)    /* 选中态 */
```

**优势**：浅色主题下自动反转（黑色叠加），一套变量适配双主题。

### 5.2 三层 Token 隔离

```
Primitive (p-) → Semantic (s-) → Component (c-)
```
每个组件模块有独立变量空间（`--ai-card-*` / `--tv-empty-*` / `--smc-panel-*`），不污染全局。

### 5.3 Web Worker 计算池

`AdvancedIndicatorWorkerPool` 通过 Web Worker 将指标计算移至后台线程，主线程仅负责渲染。相比我们的主线程计算，在大数据量场景下有显著性能优势。

### 5.4 数据源三态管理

数据源状态分三级：`normal`（正常）→ `degraded`（降级）→ `fallback`（回退），用户可直观了解数据质量。

### 5.5 AI 信号总线

`AiSignalBus` 将 AI 分析结果以事件驱动方式广播到各模块，实现"AI 分析 → 信号更新 → 自选监控 → 共振评分"的自动联动。
