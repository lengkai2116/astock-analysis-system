## 1. 设计理念与风格定位

### 1.1 设计理念

| 理念 | 说明 |
|------|------|
| **专业可信** | 面向投资者的专业工具，传递信任感 |
| **数据驱动** | 以数据可视化为主，图表清晰直观 |
| **简洁高效** | 减少认知负担，快速获取关键信息 |
| **A股特色** | 融入A股特有的交易规则和数据指标 |

### 1.2 风格定位

**配色方案**：
- **主色调**：深蓝色系（#0A1628）- 代表专业与信任
- **辅助色**：绿色（#22C55E）- 表示上涨/买入信号
- **警告色**：红色（#EF4444）- 表示下跌/卖出信号
- **中性色**：深灰（#1E293B）- 作为背景
- **高亮色**：金色（#F59E0B）- 强调重要数据

**字体方案**：
- **标题字体**：Inter（现代简洁）
- **数据字体**：JetBrains Mono（等宽字体，适合数字显示）
- **正文字体**：PingFang SC（中文阅读舒适）

### 1.3 与QuantDinger的风格差异

| 维度 | QuantDinger | 我们的系统 |
|------|------------|----------|
| **配色** | 科技蓝+紫色渐变 | 深蓝+绿/红信号色 |
| **图标风格** | 线性图标 | 金融风格图标 |
| **图表风格** | 深色主题 | 专业金融K线风格 |
| **品牌感** | 通用量化平台 | A股专业分析平台 |
| **交易功能** | 支持实盘交易 | 仅分析决策支持 |

### 5.2 SignalBadge组件实现

```vue
<!-- src/components/SignalBadge.vue -->
<template>
  <span 
    :class="['signal-badge', `signal-${signalType}`]"
    :style="{ backgroundColor: bgColor, color: textColor }"
  >
    <span class="signal-icon">{{ icon }}</span>
    <span class="signal-text">{{ text }}</span>
    <span v-if="confidence !== null" class="signal-confidence">{{ confidence }}%</span>
  </span>
</template>

<script setup>
import { computed } from 'vue'

const props = defineProps({
  signalType: {
    type: String,
    required: true,
    validator: (v) => ['buy', 'sell', 'hold', 'warning'].includes(v)
  },
  confidence: {
    type: Number,
    default: null
  }
})

const icon = computed(() => ({
  buy: '✅',
  sell: '🔴',
  hold: '⏳',
  warning: '⚠️'
}[props.signalType]))

const bgColor = computed(() => ({
  buy: 'rgba(34, 197, 94, 0.15)',
  sell: 'rgba(239, 68, 68, 0.15)',
  hold: 'rgba(148, 163, 184, 0.15)',
  warning: 'rgba(251, 191, 36, 0.15)'
}[props.signalType]))

const textColor = computed(() => ({
  buy: '#22C55E',
  sell: '#EF4444',
  hold: '#94A3B8',
  warning: '#FBBF24'
}[props.signalType]))

const text = computed(() => ({
  buy: '买入',
  sell: '卖出',
  hold: '观望',
  warning: '预警'
}[props.signalType]))
</script>
```

### 5.3 StockCard组件实现

```vue
<!-- src/components/StockCard.vue -->
<template>
  <div class="stock-card" @click="$emit('click')">
    <div class="stock-header">
      <span class="stock-code">{{ stock.code }}</span>
      <SignalBadge v-if="signal" :signal-type="signal.type" :confidence="signal.confidence" />
    </div>
    <div class="stock-name">{{ stock.name }}</div>
    <div class="stock-price" :class="{ 'price-up': stock.change >= 0, 'price-down': stock.change < 0 }">
      ¥{{ stock.price.toFixed(2) }}
    </div>
    <div class="stock-change" :class="{ 'change-up': stock.change >= 0, 'change-down': stock.change < 0 }">
      {{ stock.change >= 0 ? '+' : '' }}{{ stock.change.toFixed(2) }}%
    </div>
    <div class="stock-volume">
      成交量: {{ formatVolume(stock.volume) }}
    </div>
  </div>
</template>

<script setup>
import SignalBadge from './SignalBadge.vue'

defineProps({
  stock: {
    type: Object,
    required: true,
    default: () => ({
      code: '',
      name: '',
      price: 0,
      change: 0,
      volume: 0
    })
  },
  signal: {
    type: Object,
    default: null
  }
})

defineEmits(['click'])

const formatVolume = (volume) => {
  if (volume >= 100000000) {
    return (volume / 100000000).toFixed(2) + '亿'
  } else if (volume >= 10000) {
    return (volume / 10000).toFixed(2) + '万'
  }
  return volume.toString()
}
</script>
```
