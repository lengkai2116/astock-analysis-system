### 3.2 涨跌停限制

```python
def can_buy(self, ts_code: str, price: float, 
            limit_up_price: float, ...) -> bool:
    """检查是否能买入 - 涨停不能买"""
    if price >= limit_up_price:
        return False
    return True

def can_sell(self, ts_code: str, price: float,
            ..., limit_down_price: float) -> bool:
    """检查是否能卖出 - 跌停不能卖"""
    if price <= limit_down_price:
        return False
    return True
```

### 3.3 成本计算

```python
def calculate_order_cost(self, price: float, quantity: int, 
                        is_buy: bool = True) -> Tuple[...]:
    """计算订单成本"""
    amount = price * quantity
    
    # 手续费（买入卖出都收取）
    commission = max(amount * self.config.commission_rate, 
                   self.config.min_commission)
    
    # 印花税（仅卖出收取）
    stamp_duty = 0.0
    if not is_buy:
        stamp_duty = amount * self.config.stamp_duty_rate
    
    # 滑点（双向收取）
    slippage = amount * self.config.slippage_rate
    
    return amount, commission, stamp_duty, slippage
```

## 4. 绩效指标体系

### 4.1 基础指标

- **总收益率**: `(最终净值 - 初始净值) / 初始净值`
- **年化收益率**: `(1 + 总收益率) ^ (252/交易天数) - 1`
- **波动率**: `日收益率标准差 * √252`
- **最大回撤**: `max((最高净值 - 当前净值) / 最高净值)`

### 4.2 风险调整指标

- **夏普比率**: `√252 * (日均超额收益 / 日收益标准差)`
- **索提诺比率**: `√252 * (日均超额收益 / 下行标准差)`

### 4.3 交易统计

- **胜率**: `盈利天数 / 总交易天数`
- **盈亏比**: `平均盈利 / |平均亏损|`
- **总交易次数**: 买入 + 卖出次数
- **手续费总额**: 所有交易的手续费合计
- **印花税总额**: 所有卖出的印花税合计

### 4.4 基准对比

- **超额收益**: 策略收益 - 基准收益
- **Beta**: `cov(策略收益, 基准收益) / var(基准收益)`
- **跟踪误差**: `(策略收益 - 基准收益)标准差 * √252`
- **信息比率**: `超额收益 / 跟踪误差`
