## 第一部分：系统概述

### 1.1 项目背景

根据《A股分析系统审查与改进建议评估报告》，策略编辑器体验优化是P1优先级任务。策略模板系统作为策略编辑器的基础功能，将为用户提供：
- 预置常用策略模板
- 自定义策略模板管理
- 策略模板市场

### 1.2 核心功能

1. **策略模板库**
   - 预置常用策略（均值回归、趋势跟踪、套利等）
   - 策略分类管理
   - 策略搜索和筛选

2. **模板编辑器**
   - 可视化模板配置
   - 参数模板定义
   - 模板预览

3. **模板市场**（后期）
   - 用户模板分享
   - 模板评分和评论
   - 模板排行榜

### 3.2 示例模板：移动平均线策略

```python
"""
移动平均线策略模板
类型: 趋势跟踪
描述: 使用短期和长期移动平均线的交叉来判断趋势
"""

# 参数配置
MA_SHORT = 5   # 短期均线周期
MA_LONG = 20   # 长期均线周期
POSITION_SIZE = 100  # 每次交易数量

def initialize(context):
    """初始化策略参数"""
    context.short_ma = MA_SHORT
    context.long_ma = MA_LONG
    context.position_size = POSITION_SIZE
    
def handle_bar(context, bar):
    """处理每个K线"""
    # 计算移动平均线
    short_ma = bar.close[-context.short_ma:].mean()
    long_ma = bar.close[-context.long_ma:].mean()
    
    # 获取持仓
    position = context.portfolio.position
    
    # 交易逻辑
    if short_ma > long_ma and position == 0:
        # 金叉，买入信号
        context.buy(context.stock, context.position_size)
    elif short_ma < long_ma and position > 0:
        # 死叉，卖出信号
        context.sell(context.stock, position)
```

### 3.3 示例模板：RSI均值回归策略

```python
"""
RSI均值回归策略模板
类型: 均值回归
描述: 使用RSI指标识别超买超卖信号
"""

# 参数配置
RSI_PERIOD = 14      # RSI计算周期
RSI_OVERSOLD = 30     # 超卖阈值
RSI_OVERBOUGHT = 70   # 超买阈值
POSITION_SIZE = 100   # 每次交易数量

def initialize(context):
    """初始化策略参数"""
    context.rsi_period = RSI_PERIOD
    context.rsi_oversold = RSI_OVERSOLD
    context.rsi_overbought = RSI_OVERBOUGHT
    context.position_size = POSITION_SIZE
    
def calculate_rsi(prices, period):
    """计算RSI指标"""
    deltas = np.diff(prices)
    gains = np.where(deltas > 0, deltas, 0)
    losses = np.where(deltas < 0, -deltas, 0)
    
    avg_gain = gains[-period:].mean()
    avg_loss = losses[-period:].mean()
    
    if avg_loss == 0:
        return 100
    
    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))
    return rsi

def handle_bar(context, bar):
    """处理每个K线"""
    # 计算RSI
    rsi = calculate_rsi(bar.close, context.rsi_period)
    
    # 获取持仓
    position = context.portfolio.position
    
    # 交易逻辑
    if rsi < context.rsi_oversold and position == 0:
        # 超卖，买入信号
        context.buy(context.stock, context.position_size)
    elif rsi > context.rsi_overbought and position > 0:
        # 超买，卖出信号
        context.sell(context.stock, position)
```
