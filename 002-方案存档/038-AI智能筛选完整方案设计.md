# 达尔文筛选系统 - AI智能筛选完整方案

## 📊 方案概述

### 目标
将筛选流程分为两个独立阶段：
1. **数据准备阶段**：完成所有历史数据的获取和缓存
2. **AI筛选阶段**：使用AI模型进行智能筛选和分析

### 优势
- ✅ 数据与逻辑分离，便于迭代优化
- ✅ AI模型可进行深度分析和推理
- ✅ 支持多种AI模型（本地模型、DeepSeek API）
- ✅ 筛选标准可灵活调整，无需重新获取数据

## 🤖 第二阶段：AI智能筛选

### AI模型选择

#### 方案A：本地大模型
- **优势**：免费、隐私保护、无网络依赖
- **劣势**：需要本地部署、计算资源要求高
- **推荐模型**：
  - Qwen2.5-7B-Instruct
  - Llama3-8B-Instruct
  - Mistral-7B-Instruct

#### 方案B：DeepSeek API
- **优势**：性能强大、无需本地部署、成本低
- **劣势**：需要网络、API调用费用
- **推荐模型**：
  - deepseek-chat（通用对话）
  - deepseek-reasoner（深度推理）

#### 方案C：混合方案
- 本地模型：快速筛选、初步分析
- DeepSeek API：深度分析、复杂推理

### AI筛选流程设计

```python
# AI筛选流程
1. 从缓存数据库加载所有股票数据
2. 构建结构化数据摘要
3. 设计AI Prompt模板
4. 调用AI模型进行筛选
5. 解析AI返回结果
6. 生成筛选报告
```

### AI Prompt设计

#### Prompt模板示例

```
你是一位专业的投资分析师，精通达尔文主义投资理念。

任务：从以下股票数据中筛选出符合"生存能力"标准的优质股票。

筛选标准：
1. 财务稳健性：资产负债率合理、现金流充足
2. 盈利能力：ROE稳定、利润质量高
3. 风险控制：无ST风险、商誉占比合理

股票数据：
{stock_data_json}

请分析每只股票，并给出：
1. 是否通过筛选（通过/淘汰）
2. 风险评分（0-100）
3. 主要风险点
4. 投资建议

输出格式：JSON
```

## 🔧 技术实现

### 1. 数据获取脚本

**文件**：`fetch_complete_data.py`

**功能**：
- 获取全市场股票列表
- 批量获取财务数据
- 获取股票基本信息
- 获取历史分红数据
- 保存到缓存数据库

**配置**：
- Tushare Token：已配置
- 限流：400次/分钟
- 批次大小：100只/批
- 批次间隔：60秒

### 2. AI筛选脚本

**文件**：`ai_intelligent_filter.py`

**功能**：
- 从缓存加载数据
- 构建AI Prompt
- 调用AI模型（本地或API）
- 解析筛选结果
- 生成分析报告

**配置**：
- AI模型选择：本地/DeepSeek/混合
- Prompt模板：可自定义
- 输出格式：JSON/Markdown

### 3. 结果分析脚本

**文件**：`analyze_filter_results.py`

**功能**：
- 统计筛选结果
- 生成可视化报告
- 导出候选股票列表
- 对比不同筛选标准结果
