### 二、Level2数据权限说明

**❌ 当前阶段不必须具备Level2数据权限**（详见评估报告：[Level2数据权限必要性评估.md](file:///Users/kalence/Desktop/01-A股股票分析系统/docs/Level2数据权限必要性评估.md)）

| 权限类型 | 说明 | 当前必要性 |
|----------|------|-----------|
| **Level1** | 五档行情、分时成交、实时快照 | ✅ **必须** |
| **Level2** | 十档/千档盘口、逐笔成交、深度数据 | ⚠️ **非必须** |

**Level2使用建议**：
- 当前系统定位是A股分析决策支持系统，核心功能（日线/分钟线行情、技术指标、因子分析、回测、报告中心等）用Level1数据即可完成
- Level2应定位为**增强项**，不是MVP或第一阶段上线的硬依赖
- 只有当系统明确进入盘口微观结构分析、高频短线策略、逐笔成交分析时，Level2才变成强需求

### 三、Level1权限详细说明

**✅ **QMT Level1权限是开通QMT后即包含，无需额外申请**

#### Level1包含的行情数据

| 数据类型 | 说明 | 刷新频率 |
|----------|------|----------|

| **实时快照** | 最新成交价、涨跌幅、成交量、成交额 | 实时 |
| **分时走势** | 当天分时成交数据（合并成交） | 实时 |
| **K线数据** | 1分钟、5分钟、15分钟、30分钟、60分钟、日线、周线、月线 | 实时生成 |
| **指数行情** | 上证指数、深证成指、创业板指等指数实时行情 | 实时 |
| **全推行情** | 支持全推订阅（推荐用于多股票监控） | 实时推送 |

#### Level1包含的功能权限

| 功能类型 | 说明 |
|----------|
| **历史K线数据** | 支持获取历史K线（日线、分钟线数据 |

| **合约基础信息** | 股票、ETF、可转债等基础信息 |
| **板块和行业分类** | 行业分类、概念板块数据 |
| **实时订阅功能** | 支持实时行情订阅和推送回调 |

#### QMT核心功能（Level1已包含）

| 功能模块 | 说明 |
|----------|------|
| **策略开发** | Python/VBA双语言支持 |
| **高速回测** | 本地历史回测引擎 |
| **实盘交易** | 股票、可转债交易功能（需交易权限） |
| **数据中心** | 支持下载历史行情和财务数据 |
| **组合篮子交易** | 支持组合交易 |

**刷新频率**：期货0.5秒刷新，股票行情最快2秒刷新

**Level1与交易所标准定义**：Level-1行情是交易所向全市场提供的、最低规格的行情数据服务。包含股票最新成交价、涨跌幅、成交量、成交额、五档挂单和分时走势的实时数据。

### 四、开通流程

#### 步骤1：确认账户条件
- ✅ 招商证券股票账户（普通账户或信用账户均可）
- ✅ 连续20个交易日日均资产达标（50-300万元，具体咨询客户经理）
- ✅ 风险测评等级C4及以上

#### 步骤2：线上申请（佣金宝APP）
1. 登录**佣金宝APP**
2. 点击右下角「我的」
3. 进入「我的业务」→「权限开通」
4. 找到「PTrade/QMT/底仓增强权限」
5. 选择QMT（可选择 **MiniQMT** 或 **QMT全功能版**）
6. 填写申请信息（确保邮箱填写正确）
7. 签署《量化交易风险揭示书》
8. 完成程序化交易问卷报备
9. 确认信息无误后提交申请

> **建议**：提交申请前提前联系客户经理，告知开通意向

#### 步骤3：等待审核
- 审核时间：通常1-2个工作日
- 审核期间可主动申请测试账号和模拟环境，提前熟悉操作

#### 步骤4：开通成功通知
- 审核通过后，会收到短信或APP推送通知
- 客户经理会联系并提供miniQMT下载链接

#### 步骤5：下载安装miniQMT
- 使用客户经理提供的链接下载安装包
- 安装miniQMT客户端
- 登录并配置行情服务器

#### 步骤6：权限验证
- 登录miniQMT后，验证Level1行情数据是否正常
- 测试实时行情订阅功能
- 确认数据接口可用

#### 4.1 QMT数据提供者类

**文件**：`backend/app/data/qmt_provider.py`

```python
"""
QMT行情数据提供者 - 集成招商证券miniQMT实时行情
"""
import time
import threading
from typing import List, Dict, Optional, Callable
from datetime import datetime

try:
    from xtquant import xtdata
    QMT_AVAILABLE = True
except ImportError:
    QMT_AVAILABLE = False


class QmtDataProvider:
    """QMT行情数据提供者"""
    
    def __init__(self):
        self._callbacks = {}
        self._seq_map = {}
        self._running = False
        self._thread = None
        self._qmt_connected = False
        
    def connect(self) -> bool:
        """连接到miniQMT"""
        if not QMT_AVAILABLE:
            print("❌ xtquant库未安装")
            return False
            
        try:
            # 检查miniQMT连接状态
            xtdata.connect()
            self._qmt_connected = True
            print("✅ 成功连接到miniQMT")
            return True
        except Exception as e:
            print(f"❌ 连接miniQMT失败: {e}")
            return False
    
    def disconnect(self):
        """断开连接"""
        self._running = False
        if self._thread:
            self._thread.join(timeout=5)
        
        # 取消所有订阅
        for seq in list(self._seq_map.keys()):
            self.unsubscribe(seq)
        
        self._qmt_connected = False
        print("✅ 已断开miniQMT连接")
    
    def subscribe_tick(self, stock_codes: List[str], callback: Callable) -> int:
        """
        订阅Tick数据
        
        Args:
            stock_codes: 股票代码列表，格式如 ['600000.SH', '000001.SZ']
            callback: 回调函数，签名: callback(datas: Dict)
        
        Returns:
            订阅号，失败返回-1
        """
        if not self._qmt_connected:
            print("❌ 请先连接miniQMT")
            return -1
        
        try:
            seq = xtdata.subscribe_whole_quote(
                code_list=stock_codes,
                callback=callback
            )
            if seq > 0:
                self._seq_map[seq] = {'type': 'tick', 'codes': stock_codes, 'callback': callback}
                print(f"✅ 成功订阅Tick数据，订阅号: {seq}")
            return seq
        except Exception as e:
            print(f"❌ 订阅失败: {e}")
            return -1
    
    def subscribe_kline(self, stock_codes: List[str], period: str = '1m', 
                       callback: Optional[Callable] = None) -> Dict[str, int]:
        """
        订阅K线数据
        
        Args:
            stock_codes: 股票代码列表
            period: K线周期，支持 'tick', '1m', '5m', '15m', '30m', '1h', '1d'
            callback: 回调函数
            
        Returns:
            {stock_code: 订阅号}
        """
        if not self._qmt_connected:
            print("❌ 请先连接miniQMT")
            return {}
        
        results = {}
        for code in stock_codes:
            try:
                seq = xtdata.subscribe_quote(
                    stock_code=code,
                    period=period,
                    count=0,
                    callback=callback
                )
                if seq > 0:
                    results[code] = seq
                    self._seq_map[seq] = {'type': 'kline', 'code': code, 'period': period}
                    print(f"✅ 成功订阅 {code} {period} K线，订阅号: {seq}")
            except Exception as e:
                print(f"❌ 订阅 {code} 失败: {e}")
                results[code] = -1
        
        return results
    
    def unsubscribe(self, seq: int):
        """取消订阅"""
        try:
            xtdata.unsubscribe_quote(seq)
            if seq in self._seq_map:
                del self._seq_map[seq]
            print(f"✅ 已取消订阅，订阅号: {seq}")
        except Exception as e:
            print(f"❌ 取消订阅失败: {e}")
    
    def get_tick(self, stock_codes: List[str]) -> Dict:
        """
        获取最新Tick数据
        
        Args:
            stock_codes: 股票代码列表
            
        Returns:
            {stock_code: tick_data}
        """
        if not self._qmt_connected:
            return {}
        
        try:
            data = xtdata.get_full_tick(stock_codes)
            return data
        except Exception as e:
            print(f"❌ 获取Tick数据失败: {e}")
            return {}
    
    def get_kline(self, stock_code: str, period: str = '1d', 
                 start_time: str = '', end_time: str = '', count: int = -1) -> Optional[Dict]:
        """
        获取历史K线数据
        
        Args:
            stock_code: 股票代码
            period: 周期
            start_time: 起始时间，格式 'YYYYMMDD'
            end_time: 结束时间，格式 'YYYYMMDD'
            count: 返回条数，-1表示全部
            
        Returns:
            K线数据字典
        """
        if not self._qmt_connected:
            return None
        
        try:
            data = xtdata.get_market_data(
                stock_code=stock_code,
                period=period,
                start_time=start_time,
                end_time=end_time,
                count=count
            )
            return data
        except Exception as e:
            print(f"❌ 获取K线数据失败: {e}")
            return None
    
    def get_market_snapshot(self, stock_codes: List[str]) -> List[Dict]:
        """
        获取市场快照（最新报价）
        
        Args:
            stock_codes: 股票代码列表
            
        Returns:
            快照数据列表
        """
        if not self._qmt_connected:
            return []
        
        try:
            tick_data = self.get_tick(stock_codes)
            results = []
            
            for code, data in tick_data.items():
                results.append({
                    'ts_code': code,
                    'price': data.get('lastPrice', 0),
                    'open': data.get('openPrice', 0),
                    'high': data.get('highPrice', 0),
                    'low': data.get('lowPrice', 0),
                    'prev_close': data.get('preClosePrice', 0),
                    'volume': data.get('volume', 0),
                    'amount': data.get('turnover', 0),
                    'bid_price': data.get('bidPrice1', 0),
                    'ask_price': data.get('askPrice1', 0),
                    'bid_volume': data.get('bidVolume1', 0),
                    'ask_volume': data.get('askVolume1', 0),
                    'timestamp': datetime.now().isoformat()
                })
            
            return results
        except Exception as e:
            print(f"❌ 获取市场快照失败: {e}")
            return []
    
    def run(self, background: bool = True):
        """
        启动行情接收循环
        
        Args:
            background: 是否在后台线程运行
        """
        if not self._qmt_connected:
            print("❌ 请先连接miniQMT")
            return
        
        self._running = True
        
        if background:
            self._thread = threading.Thread(target=self._run_loop, daemon=True)
            self._thread.start()
            print("✅ 行情接收线程已启动")
        else:
            self._run_loop()
    
    def _run_loop(self):
        """内部运行循环"""
        try:
            xtdata.run()
        except Exception as e:
            print(f"❌ 行情循环异常: {e}")
        finally:
            self._running = False


# ============ 使用示例 ============
if __name__ == '__main__':
    # 创建提供者
    provider = QmtDataProvider()
    
    # 连接miniQMT
    if provider.connect():
        # 获取市场快照
        snapshot = provider.get_market_snapshot(['600519.SH', '000001.SZ'])
        for stock in snapshot:
            print(f"{stock['ts_code']}: {stock['price']}")
        
        # 订阅实时行情
        def on_tick(datas):
            for code, data in datas.items():
                print(f"[{datetime.now()}] {code}: {data.get('lastPrice')}")
        
        seq = provider.subscribe_tick(['600519.SH', '000001.SZ'], on_tick)
        
        # 启动行情循环
        provider.run(background=False)
```

```python
"""
QMT行情API路由
提供实时行情数据接口
"""
from flask import Blueprint, jsonify, request
from app.data.qmt_provider import QmtDataProvider
import threading

qmt_bp = Blueprint('qmt', __name__, url_prefix='/api/v3/qmt')

qmt_provider = None
qmt_lock = threading.Lock()


def get_qmt_provider():
    """获取QMT提供者（单例）"""
    global qmt_provider
    with qmt_lock:
        if qmt_provider is None:
            qmt_provider = QmtDataProvider()
    return qmt_provider


@qmt_bp.route('/connect', methods=['POST'])
def connect_qmt():
    """连接miniQMT"""
    provider = get_qmt_provider()
    success = provider.connect()
    return jsonify({
        'success': success,
        'message': '连接成功' if success else '连接失败，请确保miniQMT已启动并登录'
    })


@qmt_bp.route('/disconnect', methods=['POST'])
def disconnect_qmt():
    """断开miniQMT连接"""
    provider = get_qmt_provider()
    provider.disconnect()
    return jsonify({'success': True, 'message': '已断开连接'})


@qmt_bp.route('/snapshot', methods=['GET'])
def get_snapshot():
    """获取市场快照"""
    provider = get_qmt_provider()
    
    codes = request.args.get('codes', '')
    if codes:
        stock_codes = [c.strip() for c in codes.split(',')]
    else:
        stock_codes = ['600519.SH', '000001.SZ', '000002.SZ']
    
    snapshot = provider.get_market_snapshot(stock_codes)
    return jsonify({'success': True, 'data': snapshot})


@qmt_bp.route('/tick', methods=['GET'])
def get_tick():
    """获取Tick数据"""
    provider = get_qmt_provider()
    
    codes = request.args.get('codes', '')
    if codes:
        stock_codes = [c.strip() for c in codes.split(',')]
    else:
        stock_codes = ['600519.SH']
    
    tick_data = provider.get_tick(stock_codes)
    return jsonify({'success': True, 'data': tick_data})


@qmt_bp.route('/kline/<ts_code>', methods=['GET'])
def get_kline(ts_code):
    """获取K线数据"""
    provider = get_qmt_provider()
    
    period = request.args.get('period', '1d')
    count = request.args.get('count', 100, type=int)
    
    kline = provider.get_kline(ts_code, period=period, count=count)
    if kline:
        return jsonify({'success': True, 'data': kline})
    else:
        return jsonify({'success': False, 'message': '获取K线数据失败'}), 500


@qmt_bp.route('/subscribe', methods=['POST'])
def subscribe():
    """订阅实时行情"""
    provider = get_qmt_provider()
    
    data = request.get_json()
    stock_codes = data.get('codes', [])
    period = data.get('period', 'tick')
    
    if not stock_codes:
        return jsonify({'success': False, 'message': '请提供股票代码'}), 400
    
    if period == 'tick':
        seq = provider.subscribe_tick(stock_codes, _default_callback)
        return jsonify({'success': seq > 0, 'subscription_id': seq})
    else:
        results = provider.subscribe_kline(stock_codes, period=period)
        return jsonify({'success': True, 'data': results})


def _default_callback(datas):
    """默认回调函数"""
    print(f"收到行情数据: {len(datas)} 只股票")
```

### 九、数据源优先级策略

根据[Level2数据权限必要性评估报告](file:///Users/kalence/Desktop/01-A股股票分析系统/docs/Level2数据权限必要性评估.md)，推荐以下优先级：

```text
第一优先级：日线/复权/daily_basic/资金流 + QMT普通实时行情（Level1）
第二优先级：分钟线、实时快照、五档盘口展示

```

**当前系统功能对Level1的依赖验证**：

| 功能模块 | Level1是否足够 | 说明 |
|----------|--------------|------|
| 股票列表、基础资料 | ✅ 足够 | 基础数据源即可 |
| 日线K线、技术指标 | ✅ 足够 | OHLCV数据足够 |
| 因子库与多因子筛选 | ✅ 足够 | 当前因子基于日线/技术/量价 |
| 策略模板与策略输出 | ✅ 足够 | 研究和信号输出，非盘口交易 |
| A股增强回测 | ✅ 足够 | 回测粒度是日线/信号级 |
| 报告中心、AI分析 | ✅ 足够 | 结构化行情、指标和策略结果 |
| 自选股实时监控 | ✅ 足够 | Level1快照足够 |
| 实时价格推送 | ✅ 足够 | QMT Level1快照可满足 |
| 筹码分布估算 | ✅ 足够 | 可用OHLCV + 换手率估算 |
| 主力阶段识别 | ✅ 足够 | 筹码估算、量价、资金流向 |

### 十、Level2预留设计

虽然当前不需要Level2，但系统设计应预留扩展能力：

**配置开关预留**：
```text
REALTIME_PROVIDER=qmt
ENABLE_LEVEL2=false
LEVEL2_PROVIDER=
```

**数据模型预留字段**：
- bid/ask五档或十档数据
- tick序列
- 主动买卖方向
- order imbalance（盘口不平衡）
- large order flow（大单流向）
- cancel ratio（撤单率）

**何时Level2会变成必须**：
- 逐笔成交和逐笔委托回放
- 主动买入/主动卖出成交强度
- 大单拆单、扫单、压单、托单识别
- 撤单率、挂单生命周期、虚假挂单识别
- 前五档/十档盘口不平衡因子
- 竞价撮合阶段微观结构策略
- 1分钟以内的超短线或高频策略
- 实盘自动交易中的盘口择价、排队、滑点控制


### 十二、注意事项

1. **miniQMT必须保持运行**
   - 需要先启动miniQMT客户端
   - 必须登录招商证券账户
   - 建议设置为开机自启

2. **网络稳定性**
   - QMT使用XTP协议，需要稳定网络
   - 建议使用券商专用行情服务器

3. **权限管理**
   - **Level2数据当前非必须**（详见评估报告）
   - Level1权限开通即有，无需额外申请
   - Level2需单独申请且额外付费，建议后期按需开通

4. **线程安全**
   - QMT回调运行在内部线程
   - 需要注意数据同步问题

5. **数据降级策略**
   - 当QMT不可用时，自动降级到Tushare日线数据
   - 当Tushare也不可用时，降级到AkShare免费数据源

6. **前端展示**
   - 显示当前行情来源（QMT/Tushare/AkShare）
   - 显示数据更新时间
   - 显示是否为实时数据
