# A股实时行情数据源对比研究报告

## 一、概述

本报告对 A 股市场可用的实时行情数据获取渠道进行全面调研和对比。旨在为当前系统的实时数据模块（Socket.IO + 实时行情推送）选择合适的数据源，并规划后续数据升级路径。

> **调研说明**：本次调研因网络环境受限，未能进行实时在线搜索。以下内容基于截至 2026 年 5 月的公开已知信息、各平台官方文档及行业共识整理，数据源的价格和功能细节可能随平台版本更新而变化，实施前建议核实最新状态。

## 三、免费数据源详析

### 3.1 新浪财经 HTTP API

| 项目 | 说明 |
|------|------|
| URL | `https://hq.sinajs.cn/list=sh600519` |
| 返回格式 | 类 CSV 文本（`var hq_str_sh600519="..."`） |
| 延迟 | 约 3-5 秒 |
| 支持内容 | 报价、盘口五档、成交明细 |
| 并发限制 | 无明确限制，高频会封 IP |
| 适用场景 | 个人使用、低频查询 |
| 协议 | HTTP/1.1 GET |
| 费用 | 免费 |

**示例响应**：
```
var hq_str_sh600519="贵州茅台,1921.00,1922.00,1935.00,1940.00,1918.00,...
```

**优缺点**：
- ✅ 零成本
- ✅ 实现简单（一个 HTTP GET）
- ❌ 延迟较高（3-5s）
- ❌ 无认证使用可能被限流
- ❌ 非结构化文本，解析容易出错
- ❌ 无盘口深度数据（Level-2）

### Step 1 实施要点 (东方财富 WebSocket)

```python
import asyncio
import websocket
import json
import redis

class EastMoneyRealtimeProvider:
    """东方财富 WebSocket 实时行情"""

    WS_URL = "wss://push2.eastmoney.com/Push"

    def __init__(self):
        self.redis = redis.Redis(host='redis', port=6379)
        self.subscriptions = set()
        self.ws = None

    def subscribe(self, symbols: list):
        """订阅股票实时行情"""
        self.subscriptions.update(symbols)
        if self.ws and self.ws.connected:
            self._send_subscribe()

    def _send_subscribe(self):
        """发送订阅消息"""
        codes = ",".join(self.subscriptions)
        msg = json.dumps({
            "uid": "",
            "code": codes,
            "market": 1,  # 自动检测 SH/SZ
            "type": "quote"
        })
        self.ws.send(msg)

    def on_message(self, ws, message):
        """收到行情 → 推送到 Redis"""
        data = json.loads(message)
        # 格式化并发布到 Redis
        self.redis.publish('market_realtime', json.dumps({
            'ts_code': data['code'],
            'price': data['price'],
            'change': data['change'],
            'change_pct': data['changePct'],
            'timestamp': data['timestamp']
        }))
```
