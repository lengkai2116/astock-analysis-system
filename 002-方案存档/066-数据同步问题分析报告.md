## 问题分析

### 发现的问题

在 [`tushare_provider.py`](file:///Users/kalence/Desktop/测试/01-A股股票分析系统/backend/app/data/tushare_provider.py#L68) 中，`get_daily_data`方法默认只获取最近一年的数据：

```python
if start_date is None:
    start_date = (datetime.now() - pd.Timedelta(days=365)).strftime('%Y%m%d')
```

### 为什么是242条？

一年有365天，但只有约240-250个交易日（扣除周末和节假日），所以只获取到242条数据是正常的。
