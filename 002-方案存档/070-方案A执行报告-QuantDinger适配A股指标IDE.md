### 3.2 指标系统

**内置指标（12种）：**
| 指标 | 类型 | 简称 | 说明 |
|------|------|------|------|
| SMA | 主图叠加 | SMA | 简单移动平均线 |
| EMA | 主图叠加 | EMA | 指数移动平均线 |
| RSI | 副图 | RSI | 相对强弱指标 |
| MACD | 副图 | MACD | 异同移动平均线 |
| BOLL | 主图叠加 | BB | 布林带 |
| ATR | 副图 | ATR | 平均真实波幅 |
| CCI | 副图 | CCI | 商品通道指数 |
| KDJ | 副图 | KDJ | 随机指标 |

**指标特性：**
- ✅ 支持多实例（如同类型多条MA）
- ✅ 支持参数编辑（周期、乘数等）
- ✅ 支持显示/隐藏切换
- ✅ 支持颜色和线宽配置
- ✅ 自动递增默认参数

### 3.3 A股市场适配

**标的格式：**
```
沪市：600000.SH（浦发银行）、600519.SH（贵州茅台）
深市：000001.SZ（平安银行）、000002.SZ（万科A）
```

**预设自选股：**
- 平安银行 (000001.SZ)
- 万科A (000002.SZ)
- 浦发银行 (600000.SH)
- 贵州茅台 (600519.SH)
- 五粮液 (000858.SZ)
- 招商银行 (600036.SH)
- 中国平安 (601318.SH)
- 美的集团 (000333.SZ)

**K线周期：**
- 分钟：1m、5m、15m、30m
- 小时：1H、4H
- 日线：1D
- 周线：1W

**数据来源：**
- 优先：后端 API (`/api/v3/chart/kline/{symbol}`)
- 兜底：Mock 数据生成器
