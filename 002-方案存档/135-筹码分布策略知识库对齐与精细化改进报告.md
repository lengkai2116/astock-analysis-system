## 三、第二阶段增强方向（v2）

### 方向5: 资金流向集成 — 大单行为验证主力阶段 ✅ 【P0 已完成】

**实现内容**:
- `TradingPhaseDetector.detect_phase()` 新增 `moneyflow_data` 参数
- `_calc_moneyflow_score()`: 计算5日大单净额均值/方向/连续正负
- `_score_building`: 大单净额持续为正但股价不涨 → +2.0（建仓吸筹）
- `_score_washing`: 净流入转正且比例>=60% → +1.5（洗盘尾声）
- `_score_raising`: 大单净额持续为正且放大 → +2.0（拉升确认）
- `_score_shipping`: 大单净额持续为负 → +2.0（出货确认）
- `pipeline.ChipDistributionStrategy.analyze()` 在 Phase 4 前自动获取 moneyflow 数据
- 资金流向数据不可用时自动降级，不影响无钱流数据的策略运行

**涉及文件**: chip_strategy_impl.py / pipeline.py

---

### 方向6: 情绪周期辅助大盘过滤 ✅ 【P2 已完成】

**实现内容**:
- `MarketEnvironmentFilter.check_sentiment()`: 基于全市场 daily_basic 换手率构建代理情绪指标
  - 换手率<0.5% → 恐慌(x0.5)
  - 换手率0.5~1.5% → 正常(x1.0)
  - 换手率1.5~3.0% → 活跃(x1.0)
  - 换手率>3.0% → 过热(x0.7)
- `check()` 返回值新增 `sentiment` 字段（含情绪周期状态+位置乘数）
- `ChipPreFilter.filter_market()` 自动应用情绪仓位乘数
- 数据不可用时自动降级为正常

**涉及文件**: chip_pre_filter.py (MarketEnvironmentFilter)

---

### 方向7: 主力测试识别 ✅ 【P1 已完成】

**实现内容**:
- `ChipDistributionSignalGenerator._detect_mainforce_test()`:
  - 找到近N日最高价+放量点（测试动作）
  - 检测测试后缩量回落（观察盘面）
  - 判断成功：放量突破测试高点
  - 判断失败：无法突破测试高点
- 测试成功 → 强化S_BUY（仓位提升至0.9）
- 测试成功但其他条件不足 → 生成独立参与信号(0.5)
- 测试失败 → 仅强化S_WASH_END（不推高仓位）

**涉及文件**: chip_strategy_impl.py (ChipDistributionSignalGenerator)

---

### 方向8: 7种筹码形态模式匹配 ✅ 【P1 已完成】

**实现内容**:
- `ChipDistributionSignalGenerator._match_chip_pattern()`:
  1. 放量突破单峰 → S_BUY confidence +0.15
  2. 缩量回踩密集峰 → S_WASH_END confidence +0.15
  3. 缩量振荡集中 → S_BUY confidence +0.10
  4. 高点放量单峰 → S_SELL confidence +0.20
  5. 密集峰快移向上/下 → 趋势信号 +0.15
  6. 回探滞涨 → S_DIVERG_SELL +0.10
  7. 缩量上穿密集峰 → S_BUY +0.20
- 形态匹配结果同时影响仓位（position += boost）
- 使用 chip_indicators.find_peak_positions / calculate_transfer_rate 等既有接口
- 数据不足时返回空列表，不影响主流程

**涉及文件**: chip_strategy_impl.py (ChipDistributionSignalGenerator)

---

### 方向9: 洗盘结束特征增强 ✅ 【P1 已完成】

**实现内容**:
- `_check_s_wash_end()` 新增3个条件:
  - 条件5(原4升级): 均线支撑 — 价格在MA20或MA60±3%范围内
  - 条件6(新增): 黄金坑检测 — `_detect_golden_pit()`: 跌幅>=8% + 坑底缩量(<均量50%) + 反弹>=5%
  - 条件7(新增): 放量下跌转缩量企稳 — `_detect_volume_stabilization()`: 前期放量跌(>3%+量比>1.3)转缩量稳(量比<0.8+波动<2%)
- 金字塔仓位 — 满足条件数越多仓位越高

**涉及文件**: chip_strategy_impl.py (ChipDistributionSignalGenerator._check_s_wash_end)
