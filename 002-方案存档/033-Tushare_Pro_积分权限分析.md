## 三、各积分档详细对比与达尔文系统适用性

### 3.1 120积分档（免费）

**限制**:
- ❌ 只有股票日线
- ❌ 无财务报表数据
- ❌ 无分红数据
- ❌ 每天8000次上限，每分钟50次

**对达尔文系统适用性**: ❌ **完全不可用**

### 3.2 2000积分档（200元/年）⭐⭐⭐ **可选**

**包含内容**:
- ✅ 基础数据（股票列表）
- ✅ 低频行情（日、周、月）
- ✅ 财务数据（三大报表，**80次/分钟**）
- ✅ ST股票列表
- ✅ 宏观经济
- ✅ 沪港通股票列表

**限制**:
- ⚠️ 财务数据只有80次/分钟，全市场筛选时可能需要排队
- ⚠️ 缺少一些参考数据（质押、龙虎榜等）
- ⚠️ 没有特色数据

**对达尔文系统适用性**: ⚠️ **可用但受限**

**优点**:
- ✅ 便宜（200元/年）
- ✅ 基本数据都有

**缺点**:
- ⚠️ 财务数据调用频率受限（80次/分钟）
- ⚠️ 没有特色数据

### 3.3 5000积分档（500元/年）⭐⭐⭐⭐⭐ **强烈推荐**

**包含内容**:
- ✅ 2000积分档的全部内容
- ✅ 财务数据无限制（推测，因为没有特别注明80次/分）
- ✅ 次分钟数据
- ✅ 参考数据（质押、解禁、回购、增减持、龙虎榜、融资融券）
- ✅ 港股/美股/外汇基础信息
- ✅ 特色数据（概念板块、资金流向、券商金股、筹码分布、量化因子）

**优势**:
- ✅ 500次/分钟，处理速度快
- ✅ 财务数据应该没有80次/分钟的限制（明细表未特别注明）
- ✅ 有特色数据，可增强分析
- ✅ 参考数据丰富，可做更多维度分析
- ✅ 性价比高（500元/年，约42元/月）

**对达尔文系统适用性**: ✅ **完美适合**

### 3.4 10000积分档及以上（1000元/年+）

**特色数据权限**:
- 更多特色数据（盈利预测、机构调研、涨停榜单、游资数据等）

**对达尔文系统适用性**: ⭐⭐ 可选但不是必需

## 六、系统使用建议

### 6.1 使用DuckDB缓存减少API调用

```python
# 我们已有的方案
Tushare API → DuckDB缓存 → 应用层
```

这样可以：
- 减少重复请求，不浪费积分
- 提高查询速度
- 离线可用

### 6.2 API调用频率控制

即使5000积分有500次/分钟，也要：
- 合理控制并发
- 避免短时间内大量请求
- 实现请求队列和限流

### 6.3 定期更新策略

- 股票基础信息：每周更新一次
- 日线行情：每日收盘后更新
- 财务报表：每个财报季后更新
- 分红数据：分红季后更新

## 七、快速开始指南

### 7.1 注册流程

1. 访问 https://tushare.pro
2. 点击右上角"注册"
3. 填写基本信息（用户名、邮箱、密码）
4. 完成邮箱验证
5. 登录进入"个人中心"
6. 完成实名认证（必要）
7. 获取API Token

### 7.2 获取积分建议

**推荐步骤**:
1. 先注册账号并实名认证
2. 查看微信专业用户群活动（https://tushare.pro/document/1?doc_id=270）
3. 或直接捐助500元获取5000积分
4. 获取Token后配置到系统

### 7.3 Token配置

获取Token后，在系统中这样配置：

```python
import tushare as ts

# 方式1：直接设置
ts.set_token('你的Token字符串')

# 方式2：用环境变量
import os
os.environ['TUSHARE_TOKEN'] = '你的Token字符串'

# 初始化
pro = ts.pro_api()

# 测试
df = pro.stock_basic(ts_code='600519.SH', fields='ts_code,name,industry,list_date')
print(df)
```
