## 一、项目背景与目标

### 1.1 项目概述

本项目旨在构建一套完整的 **A股股票分析决策支持系统**，基于 QuantDinger 的专业量化平台架构，针对 A 股市场进行深度定制。系统专注于技术分析、指标策略开发和回测验证，**不涉及实盘交易对接**。

### 1.2 核心目标

| 目标 | 说明 |
|------|------|
| **专业分析** | 提供专业级的技术分析和量化策略开发环境 |
| **指标IDE** | 完整的指标编辑器，支持 Python 策略编写和回测 |
| **缠论支持** | 集成缠中说禅理论的技术分析功能 |
| **回测验证** | 提供完整的回测框架和结果分析 |
| **决策支持** | AI 驱动的分析建议和信号输出 |

### 1.3 与 QuantDinger 的关系

| 维度 | QuantDinger | 本系统 |
|------|-------------|--------|
| **目标市场** | 加密货币、外汇、全球股票 | A股（沪深） |
| **交易功能** | 支持实盘交易 | 仅决策支持 |
| **数据源** | 交易所 API | A股数据接口 |
| **品牌定位** | 通用量化平台 | A股专业分析 |

## 三、UI设计规范

| 原则 | 说明 |
|------|------|
| **专业可信** | 面向投资者，传递专业和信任感 |

## 五、开发优先级与计划

### 5.1 开发优先级矩阵

| 模块 | 功能完整性 | 技术难度 | 优先级 | 建议顺序 |
|------|-----------|----------|--------|----------|
| 指标IDE | 70% | 中 | P0 | 第1批 |
| 仪表盘 | 0% | 低 | P0 | 第1批 |
| 信号监控 | 0% | 低 | P0 | 第2批 |
| 缠论分析 | 0% | 高 | P1 | 第3批 |
| 回测中心 | 0% | 高 | P1 | 第3批 |
| 风控中心 | 0% | 中 | P1 | 第2批 |
| AI分析 | 0% | 高 | P2 | 第4批 |
| 报告中心 | 0% | 低 | P2 | 第4批 |

### 5.2 分阶段开发计划

#### 第一阶段：核心框架（1-2周）
1. **完善指标IDE** - 补全指标编辑、参数调整、Python执行
2. **开发仪表盘** - 市场概览、自选股监控
3. **基础组件库** - SignalBadge、StockCard 等通用组件

#### 第二阶段：分析功能（2-3周）
1. **信号监控** - 信号列表、筛选、详情
2. **风控中心** - 仓位计算、止损止盈
3. **缠论分析** - 分型识别、笔分析

#### 第三阶段：高级功能（3-4周）
1. **回测中心** - 回测配置、执行、结果展示
2. **缠论增强** - 中枢识别、背驰判断
3. **AI分析助手** - 智能问答、分析报告

#### 第四阶段：完善优化（1-2周）
1. **报告中心** - 报告生成、导出
2. **性能优化** - 图表渲染、数据加载
3. **UI优化** - 动画、交互细节
