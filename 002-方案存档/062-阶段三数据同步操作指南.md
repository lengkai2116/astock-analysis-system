### 步骤2：同步日线数据

进入后端容器并运行数据同步脚本：

```bash
# 方式一：进入容器执行

cd /app
python sync_data.py

# 方式二：直接执行
docker exec -w /app 01-a-backend-1 python sync_data.py
```

**注意**：数据同步可能需要较长时间，取决于网络状况和Tushare API限制。

## 技术指标说明

系统支持以下技术指标计算：

### 移动平均线（MA）
- MA5：5日简单移动平均
- MA10：10日简单移动平均
- MA20：20日简单移动平均

### MACD指标
- DIF：快线（EMA12 - EMA26）
- DEA：慢线（EMA(DIF, 9)）
- HIST：柱状图（2 × (DIF - DEA)）

### RSI指标
- RSI14：14日相对强弱指数

### KDJ指标
- K：随机K值
- D：随机D值
- J：随机J值（3×K - 2×D）

### BOLL指标
- BOLL_UPPER：布林线上轨
- BOLL_MID：布林线中轨
- BOLL_LOWER：布林线下轨

### 成交量指标
- VOL_MA5：5日平均成交量
- VOL_MA10：10日平均成交量

## 常见问题

### Q1: 数据同步时间过长怎么办？

**原因**：Tushare API有访问频率限制，网络状况不佳

**解决方案**：
1. 检查网络连接
2. 等待API限流恢复
3. 分批同步数据

### Q2: 技术指标计算失败？

**检查项**：
1. 确认日线数据已同步（`SELECT COUNT(*) FROM daily_data;`）
2. 确认日期范围正确
3. 检查后端日志：`docker logs 01-a-backend-1`

### Q3: Tushare API调用失败？

**可能原因**：
1. Token无效或过期
2. 积分不足（需要5000积分）
3. API调用超限

**解决方案**：
1. 登录Tushare Pro检查Token有效性
2. 确保账户积分≥5000
3. 等待限流恢复或联系Tushare客服
