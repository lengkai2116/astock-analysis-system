## 三、启动指南

### 3.1 启动后端服务

#### 步骤 1: 准备环境
```bash
cd backend
```

#### 步骤 2: 配置API Key (可选)
编辑 `.env` 文件，添加DeepSeek API Key（如果需要使用AI功能）：
```bash
DEEPSEEK_API_KEY=sk-your-actual-key-here
LLM_PROVIDER=mock  # 或 deepseek, lm_studio
```

#### 步骤 3: 启动后端服务
```bash
python3 run.py
```

后端将在 **http://localhost:5000** 启动

---

### 3.2 启动前端服务

#### 步骤 1: 进入前端目录
```bash
cd frontend/vue-project
```

#### 步骤 2: 安装依赖 (如果首次启动)
```bash
npm install
```

#### 步骤 3: 启动前端服务
```bash
npm run dev
```

前端将在 **http://localhost:5173** (或其他端口)启动

---

### 3.3 数据库初始化 (可选)

如果需要初始化预设策略模板：
```bash
cd backend
python3 -m app.services.init_system_templates
```

## 六、DeepSeek AI功能配置

### 6.1 配置模式
系统支持3种LLM模式：

| 模式 | `LLM_PROVIDER` | 说明 |
|-----|-------------|-----|
| **Mock** | `mock` | 模拟AI输出（默认，无需API Key） |
| **DeepSeek** | `deepseek` | DeepSeek官方API |
| **LM Studio** | `lm_studio` | 本地大模型 |

### 6.2 启用DeepSeek
1. 获取API Key: https://platform.deepseek.com/
2. 编辑 `.env`:
```bash
DEEPSEEK_API_KEY=sk-xxx-xxx-xxx
LLM_PROVIDER=deepseek
```
3. 重启后端服务
