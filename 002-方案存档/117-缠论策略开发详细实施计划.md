## 三、核心算法详细设计

### 3.1 线段识别算法

#### 3.1.1 算法原理

线段是由至少三笔构成，具有独立走势特征的结构。

```
笔1（↓） + 笔2（↑） + 笔3（↓） → 线段（下降）
笔1（↑） + 笔2（↓） + 笔3（↑） → 线段（上升）
```

#### 3.1.2 算法实现

```python
class SegmentAnalyzer:
    """线段分析器"""
    
    def __init__(self):
        self.min_stroke_count = 3  # 最少3笔
    
    def build_segments(self, strokes: list) -> list:
        """
        构建线段
        
        Args:
            strokes: 笔列表
        
        Returns:
            线段列表
        """
        segments = []
        
        i = 0
        while i < len(strokes) - 2:
            stroke1 = strokes[i]
            stroke2 = strokes[i + 1]
            stroke3 = strokes[i + 2]
            
            # 检查是否构成线段
            if self._is_valid_segment(stroke1, stroke2, stroke3):
                segment = self._create_segment(stroke1, stroke2, stroke3)
                segments.append(segment)
                i += 3  # 线段至少由3笔构成
            else:
                i += 1
        
        return segments
    
    def _is_valid_segment(self, s1, s2, s3) -> bool:
        """判断三笔是否构成线段"""
        # 方向检查
        if s1['direction'] == s2['direction'] or s2['direction'] == s3['direction']:
            return False
        
        # 重叠检查
        if not self._has_overlap(s1, s2, s3):
            return False
        
        # 幅度检查（简化版）
        return True
    
    def _has_overlap(self, s1, s2, s3) -> bool:
        """检查三笔是否有重叠"""
        # 简化：检查转折点是否在合理范围内
        return True
```

### 3.2 中枢识别算法

#### 3.2.1 算法原理

中枢是由至少三段构成的重叠区域，代表走势的平衡地带。

```
线段1（↑） + 线段2（↓） + 线段3（↑）
     ↓            ↓           ↓
   高点A        低点B       高点C
     └────────────┴────────────┘
              中枢区间[min(B), max(A,C)]
```

#### 3.2.2 算法实现

```python
class ZhongshuAnalyzer:
    """中枢分析器"""
    
    def __init__(self):
        self.min_segment_count = 3  # 最少3段
    
    def find_zhongshu(self, segments: list) -> list:
        """
        识别中枢
        
        Args:
            segments: 线段列表
        
        Returns:
            中枢列表
        """
        zhongshu_list = []
        
        i = 0
        while i < len(segments) - 2:
            seg1 = segments[i]
            seg2 = segments[i + 1]
            seg3 = segments[i + 2]
            
            # 检查是否构成中枢
            if self._is_valid_zhongshu(seg1, seg2, seg3):
                zhongshu = self._create_zhongshu(seg1, seg2, seg3)
                zhongshu_list.append(zhongshu)
                i += 3
            else:
                i += 1
        
        return zhongshu_list
    
    def _is_valid_zhongshu(self, seg1, seg2, seg3) -> bool:
        """判断三段是否构成中枢"""
        # 方向交替检查
        if seg1['direction'] == seg2['direction'] or seg2['direction'] == seg3['direction']:
            return False
        
        # 重叠区域检查
        overlap = self._calculate_overlap(seg1, seg2, seg3)
        return overlap is not None
    
    def _create_zhongshu(self, seg1, seg2, seg3) -> dict:
        """创建中枢结构"""
        # 计算重叠区间
        high = min(seg1['end_price'], seg3['end_price'])
        low = max(seg1['end_price'], seg3['end_price'])
        
        return {
            'start_idx': seg1['start_idx'],
            'end_idx': seg3['end_idx'],
            'high': high,
            'low': low,
            'center': (high + low) / 2,
            'range': high - low,
            'segments': [seg1, seg2, seg3],
            'direction': seg1['direction']
        }
    
    def _calculate_overlap(self, seg1, seg2, seg3) -> tuple:
        """计算三段的重叠区间"""
        # 简化实现
        return (min(seg1['end_price'], seg3['end_price']),
                max(seg1['end_price'], seg3['end_price']))
```

### 3.3 背驰判断算法

#### 3.3.1 算法原理

背驰是指走势在创新高/新低时，力度明显减弱的现象。

```
背驰类型：
1. 趋势背驰：创新高/低后，力度明显减弱
2. 盘整背驰：回调力度大于前一波段
3. 中枢破坏：离开中枢的力度小于返回的力度
```

#### 3.3.2 算法实现

```python
class DivergenceDetector:
    """背驰检测器"""
    
    def __init__(self):
        self.lookback_period = 60  # 回看周期
    
    def detect_divergence(self, zhongshu_list: list, 
                         strokes: list, 
                         volume: pd.Series = None) -> dict:
        """
        检测背驰
        
        Args:
            zhongshu_list: 中枢列表
            strokes: 笔列表
            volume: 成交量数据
        
        Returns:
            背驰分析结果
        """
        result = {
            'has_divergence': False,
            'type': None,  # 'trend', 'consolidation', 'zhongshu'
            'direction': None,
            'confidence': 0.0,
            'details': {}
        }
        
        if len(zhongshu_list) == 0 or len(strokes) < 4:
            return result
        
        # 趋势背驰检测
        trend_div = self._detect_trend_divergence(strokes)
        if trend_div['detected']:
            return trend_div
        
        # 盘整背驰检测
        consolid_div = self._detect_consolidation_divergence(strokes)
        if consolid_div['detected']:
            return consolid_div
        
        # 中枢破坏检测
        zhongshu_div = self._detect_zhongshu_divergence(zhongshu_list)
        if zhongshu_div['detected']:
            return zhongshu_div
        
        return result
    
    def _detect_trend_divergence(self, strokes: list) -> dict:
        """
        检测趋势背驰
        
        原理：创新高/低后，RSI/MACD等指标背离
        """
        if len(strokes) < 4:
            return {'detected': False}
        
        # 获取最近的上涨和下跌波段
        recent_up = [s for s in strokes if s['direction'] == 'up'][-2:]
        recent_down = [s for s in strokes if s['direction'] == 'down'][-2:]
        
        if len(recent_up) < 2 or len(recent_down) < 2:
            return {'detected': False}
        
        # 检查上涨趋势背驰
        if recent_up[-1]['end_price'] > recent_up[-2]['end_price']:
            # 价格创新高
            price_increase = recent_up[-1]['end_price'] - recent_up[-1]['start_price']
            prev_increase = recent_up[-2]['end_price'] - recent_up[-2]['start_price']
            
            # 力度减弱
            if price_increase < prev_increase * 0.8:
                return {
                    'detected': True,
                    'type': 'trend',
                    'direction': 'up',
                    'confidence': 0.8,
                    'details': {
                        'current_increase': price_increase,
                        'prev_increase': prev_increase,
                        'strength_ratio': price_increase / prev_increase
                    }
                }
        
        # 检查下跌趋势背驰（类似逻辑）
        # ...
        
        return {'detected': False}
    
    def _detect_consolidation_divergence(self, strokes: list) -> dict:
        """
        检测盘整背驰
        
        原理：回调力度大于前一波段的离开力度
        """
        if len(strokes) < 4:
            return {'detected': False}
        
        # 获取最近的结构
        last_down = strokes[-1] if strokes[-1]['direction'] == 'down' else None
        prev_up = strokes[-2] if len(strokes) >= 2 and strokes[-2]['direction'] == 'up' else None
        
        if not last_down or not prev_up:
            return {'detected': False}
        
        # 下跌幅度
        down_amplitude = last_down['end_price'] - last_down['start_price']
        prev_up_amplitude = prev_up['end_price'] - prev_up['start_price']
        
        # 回调幅度大于离开幅度 = 背驰
        if abs(down_amplitude) > abs(prev_up_amplitude) * 1.1:
            return {
                'detected': True,
                'type': 'consolidation',
                'direction': 'down',
                'confidence': 0.7,
                'details': {
                    'down_amplitude': abs(down_amplitude),
                    'prev_up_amplitude': abs(prev_up_amplitude)
                }
            }
        
        return {'detected': False}
    
    def _detect_zhongshu_divergence(self, zhongshu_list: list) -> dict:
        """
        检测中枢破坏
        
        原理：离开中枢的力度小于返回的力度
        """
        if len(zhongshu_list) < 1:
            return {'detected': False}
        
        # 简化实现
        return {'detected': False}
```

### 3.4 买卖点识别算法

#### 3.4.1 算法原理

三类买卖点是缠论中最核心的交易信号：

```
第一类买点：下跌趋势背驰的最低点
第二类买点：第一类买点后上涨回调不创新低的点
第三类买点：上涨过程中回调不进入中枢的点

第一类卖点：上涨趋势背驰的最高点
第二类卖点：第一类卖点后下跌反弹不创新高的点
第三类卖点：下跌过程中反弹不进入中枢的点
```

#### 3.4.2 算法实现

```python
class BuySellPointDetector:
    """买卖点检测器"""
    
    def __init__(self):
        self.min_confidence = 0.6  # 最小置信度
    
    def find_buy_sell_points(self, 
                             strokes: list,
                             zhongshu_list: list,
                             divergence: dict) -> tuple:
        """
        识别买卖点
        
        Returns:
            (buy_points, sell_points)
        """
        buy_points = []
        sell_points = []
        
        # 第一类买卖点：基于背驰
        if divergence['detected']:
            if divergence['direction'] == 'up':
                buy_points.append({
                    'type': 'first_buy',
                    'confidence': divergence['confidence'],
                    'position': divergence.get('position'),
                    'reason': '下跌趋势背驰'
                })
            else:
                sell_points.append({
                    'type': 'first_sell',
                    'confidence': divergence['confidence'],
                    'position': divergence.get('position'),
                    'reason': '上涨趋势背驰'
                })
        
        # 第二类买卖点：回调不创新低/高
        second_points = self._find_second_points(strokes, buy_points, sell_points)
        buy_points.extend(second_points['buy'])
        sell_points.extend(second_points['sell'])
        
        # 第三类买卖点：不进入中枢
        third_points = self._find_third_points(strokes, zhongshu_list)
        buy_points.extend(third_points['buy'])
        sell_points.extend(third_points['sell'])
        
        # 过滤低置信度
        buy_points = [p for p in buy_points if p['confidence'] >= self.min_confidence]
        sell_points = [p for p in sell_points if p['confidence'] >= self.min_confidence]
        
        return buy_points, sell_points
    
    def _find_second_points(self, strokes: list, 
                           existing_buy: list, 
                           existing_sell: list) -> dict:
        """
        识别第二类买卖点
        
        第二类买点：第一类买点后上涨，回调不创新低
        第二类卖点：第一类卖点后下跌，反弹不创新高
        """
        second_buy = []
        second_sell = []
        
        # 检查是否有第一类买点后的回调
        if existing_buy:
            first_buy = existing_buy[0]
            first_buy_idx = first_buy.get('position', {}).get('idx', -1)
            
            # 查找后续的上涨和回调
            for i, stroke in enumerate(strokes):
                if stroke.get('idx', 0) <= first_buy_idx:
                    continue
                
                if stroke['direction'] == 'up' and i < len(strokes) - 1:
                    # 检查回调是否不创新低
                    next_stroke = strokes[i + 1]
                    if next_stroke['direction'] == 'down':
                        if next_stroke['end_price'] > first_buy['position']['price']:
                            second_buy.append({
                                'type': 'second_buy',
                                'confidence': 0.7,
                                'position': {
                                    'idx': next_stroke['end_idx'],
                                    'price': next_stroke['end_price'],
                                    'date': next_stroke['end_date']
                                },
                                'reason': '回调不创新低'
                            })
        
        return {'buy': second_buy, 'sell': second_sell}
    
    def _find_third_points(self, strokes: list, zhongshu_list: list) -> dict:
        """
        识别第三类买卖点
        
        第三类买点：上涨回调不进入中枢
        第三类卖点：下跌反弹不进入中枢
        """
        third_buy = []
        third_sell = []
        
        if not zhongshu_list:
            return {'buy': third_buy, 'sell': third_sell}
        
        latest_zhongshu = zhongshu_list[-1]
        
        for i, stroke in enumerate(strokes[:-1]):
            if stroke['direction'] == 'up' and i < len(strokes) - 1:
                # 检查回调是否不进入中枢
                next_stroke = strokes[i + 1]
                if next_stroke['direction'] == 'down':
                    # 中枢上沿
                    zhongshu_high = latest_zhongshu['high']
                    
                    # 回调低点在中枢上方 = 第三类买点
                    if next_stroke['end_price'] > zhongshu_high:
                        third_buy.append({
                            'type': 'third_buy',
                            'confidence': 0.6,
                            'position': {
                                'idx': next_stroke['end_idx'],
                                'price': next_stroke['end_price'],
                                'date': next_stroke['end_date']
                            },
                            'reason': f'回调不进入中枢(中枢上沿:{zhongshu_high})'
                        })
        
        return {'buy': third_buy, 'sell': third_sell}
```
