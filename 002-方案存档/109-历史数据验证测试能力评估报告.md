## 二、技术架构

### 2.1 数据层

```
Tushare Pro API
    ↓
TushareProvider.get_daily_data()
    ↓
DataFrame (ts_code, trade_date, open, high, low, close, vol)
```

### 2.2 策略层

```
ChipDistributionStrategy.analyze()
    ↓
├── ChipDistributionService.calculate_chip_distribution()
├── ChipIndicators.calculate_all_indicators()
│   ├── SSRP (市场平均成本)
│   ├── ASR (活跃浮筹)
│   ├── CYQKL (筹码穿透力)
│   ├── RSI (相对强弱)
│   ├── 成交量指标
│   └── 筹码集中度
├── Phase Detection (5个操盘阶段)
└── Signal Generation (6种信号)
```

### 2.3 回测层

```
AShareBacktestEngine.run_backtest()
    ↓
├── 价格数据 (price_data)
├── 交易信号 (signals)
└── 绩效指标 (metrics)
    ├── total_return
    ├── annual_return
    ├── max_drawdown
    ├── sharpe_ratio
    └── win_rate
```
