## 三、技术栈总结

- **WebSocket**: Socket.IO-Client 4.7.4

## 五、运行说明

### 启动后端服务
```bash
cd backend
pip install -r requirements.txt  # 首次安装依赖
python run.py
```

服务将在 `http://localhost:5001` 启动，并自动初始化实时数据服务。

### 启动前端服务
```bash
cd frontend/vue-project
npm install  # 首次安装依赖
npm run dev
```

服务将在 `http://localhost:9001` 启动。
