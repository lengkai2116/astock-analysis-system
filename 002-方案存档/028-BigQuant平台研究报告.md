# BigQuant 平台研究报告

## 研究时间
2026-05-15

## 一、平台概述

BigQuant 是一个融合人工智能（AI）与量化投资的全栈式、研究与交易平台，其软件架构设计体现了"云原生 + 微服务 + 数据驱动 + AI 集成"的现代金融科技平台特征。

### 平台特点
- 企业级 AI 底层平台
- PB 级量化专用标准金融数据
- 2000+ 基础因子库
- 可视化开发环境
- 超参搜索、组合优化器等高级功能

## 二、SDK 研究结果

### 2.1 BigQuant SDK 安装情况

**安装包信息：**
- 包名：`bigquant`
- 版本：0.1.0
- 大小：2.0 KB

**安装方式：**
```bash
pip install bigquant -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**测试结果：**
✅ 安装成功
⚠️ 包内容非常简单（仅显示 "hello bigquant!" 消息）
❌ 不包含实际的数据获取功能
❌ 没有 DataSource 类或类似接口

### 2.2 SDK 使用限制

从测试结果来看，PyPI 上安装的 `bigquant` 包只是一个测试/演示包，**不具备实际功能**。

## 三、BigQuant 数据 API 分析

### 3.1 API 使用方式（平台内）

根据官方文档，BigQuant 在其云平台内部提供了丰富的数据 API：

**基本调用方法：**
```python
DataSource('表名').read(instruments, 
                        start_date='2005-01-01', 
                        end_date=None, 
                        fields=['字段1','字段2'...])
```

**可用数据表：**

| 表名 | 说明 |
|------|------|
| `basic_info_CN_STOCK_A` | A股基础信息 |
| `bar1d_CN_STOCK_A` | A股日线行情 |
| `stock_status_CN_STOCK_A` | 股票状态（ST、停牌等） |
| `market_value_CN_STOCK_A` | 估值分析（市值、PE、PB等） |
| `industry_CN_STOCK_A` | 行业概念 |
| `index_constituent_CN_STOCK_A` | 指数成分 |
| `net_amount_CN_STOCK_A` | 资金流数据 |
| （更多表） | （更多数据类型） |

**财务相关数据：** 文档未详细列出财务报表表名，但平台应包含此类数据

### 3.2 关键限制

⚠️ **平台专用 API** - 这些 API **只能在 BigQuant 云平台内部使用**，无法在本地环境中直接调用

## 四、对达尔文筛选系统的适用性分析

### 4.1 数据需求匹配度

| 数据需求 | BigQuant 是否提供 | 本地是否可获取 |
|---------|------------------|--------------|
| 股票列表 | ✅ 有 | ❌ 仅限平台内 |
| 财务报表 | ✅ 有（推测） | ❌ 仅限平台内 |
| 市值信息 | ✅ 有 | ❌ 仅限平台内 |
| ST股票状态 | ✅ 有 | ❌ 仅限平台内 |
| 分红数据 | ✅ 有（推测） | ❌ 仅限平台内 |
| 历史K线 | ✅ 有 | ❌ 仅限平台内 |

### 4.2 适用性结论

❌ **BigQuant 不适合作为达尔文筛选系统的数据源**

**原因：**
1. PyPI 上的 bigquant 包没有实际功能
2. 真实的数据 API 只能在 BigQuant 云平台内部使用
3. 无法在本地系统中直接获取真实数据

## 五、替代方案建议

### 方案对比

| 数据源 | 本地API | 需要注册 | 财务数据 | 推荐度 |
|--------|--------|---------|---------|--------|
| **AKShare** | ✅ 有 | ❌ 不需要 | ✅ 有 | ⭐⭐⭐⭐ |
| **Tushare Pro** | ✅ 有 | ✅ 需要 | ✅ 有 | ⭐⭐⭐⭐⭐ |
| **BaoStock** | ✅ 有 | ❌ 不需要 | ⚠️ 受限 | ⭐⭐ |
| **eFinance** | ✅ 有 | ❌ 不需要 | ❌ 无 | ⭐⭐ |
| **BigQuant** | ❌ 无 | ✅ 需要 | ✅ 有（平台内） | ⭐ |

### 推荐方案

1. **等待 AKShare 网络恢复** - 如果只是临时网络问题
2. **注册 Tushare Pro** - 最稳定、数据最全面的方案
3. **使用模拟数据继续开发** - 不依赖外部数据源

## 六、BigQuant 的用途

虽然不适合作为数据源，但 BigQuant 平台本身具有很高的研究和使用价值：

### 平台内使用场景
- ✅ 策略回测
- ✅ 因子挖掘与分析

### 使用建议
可以在 BigQuant 平台上进行策略研究和回测，同时在本地系统中使用其他数据源进行筛选。
