## 二、问题诊断与解决

### 2.1 问题一：Flask-Migrate模块缺失

**错误信息**:
```
ModuleNotFoundError: No module named 'flask_migrate'
```

**根本原因**: requirements.txt 中未包含 Flask-Migrate 依赖

**解决方案**:
```python
# 在 requirements.txt 中添加
Flask-Migrate==4.0.5
```

**验证结果**: ✅ 问题已解决，后端成功启动

---

### 2.2 问题二：PostgreSQL驱动缺失

**错误信息**:
```
ModuleNotFoundError: No module named 'psycopg2'
```

**根本原因**: psycopg2-binary 未添加到 requirements.txt

**解决方案**:
```python
# 在 requirements.txt 中添加
psycopg2-binary==2.9.9
```

**验证结果**: ✅ 问题已解决，数据库连接正常
