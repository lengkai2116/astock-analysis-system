## 第三部分：包含关系处理

### 3.1 包含关系定义（PDF第18-19页）

```
包含关系定义：
当两根相邻的K线中，一根被另一根完全包含住时，
就不符合四种分型的条件和特点，需要进行合并处理。
```

### 3.2 包含关系判断规则

```python
def has_contain_relation(kline1, kline2):
    """
    判断两根K线是否存在包含关系
    
    包含关系定义：
    - K线1的高点 <= K线2的高点 且 K线1的低点 >= K线2的低点（K线1包含在K线2内）
    - 或 K线2的高点 <= K线1的高点 且 K线2的低点 >= K线1的低点（K线2包含在K线1内）
    
    返回：True-存在包含关系，False-不存在包含关系
    """
    # K线1完全被K线2包含
    condition1 = (kline2.high >= kline1.high) and (kline2.low <= kline1.low)
    
    # K线2完全被K线1包含
    condition2 = (kline1.high >= kline2.high) and (kline1.low <= kline2.low)
    
    return condition1 or condition2
```

```

├─────────────────────────────────────────────────────────────┤
│                                                             │
│  判断依据：包含关系处理前的分型方向                          │

│  │  如果处理前走势是【下降】：                          │   │
│  │    → 取两根K线中【最低的高点】和【最低的低点】       │   │
```

### 3.4 包含关系处理代码实现

```python
def merge_contain_klines(klines, directions):
    """
    包含关系处理主函数
    
    参数：
    - klines: 原始K线列表
    - directions: 每根K线的方向列表（用于判断处理前的走势方向）
    
    返回：处理包含关系后的K线列表
    
    算法流程：
    1. 从左到右依次处理每两根相邻K线
    2. 判断是否存在包含关系
    3. 如存在包含关系，根据前一根K线的方向决定如何合并
    4. 合并后继续处理下一根K线（递归处理连续包含的情况）
    """
    if len(klines) <= 2:
        return klines
    
    result = []
    i = 0
    
    while i < len(klines):
        if i == 0:
            result.append(klines[i])
            i += 1
            continue
        
        # 获取前一根处理后的K线
        prev = result[-1]
        curr = klines[i]
        
        # 判断是否存在包含关系
        if has_contain_relation(prev, curr):
            # 获取处理前的走势方向
            # 使用前一根K线之前的方向（即当前K线处理前的方向）
            if i - 1 < len(directions):
                direction = directions[i - 1]
            else:
                direction = directions[-1] if directions else 'up'
            
            # 根据方向进行合并
            if direction == 'up':
                # 上升走势：取最高的高点和最高的低点
                merged = ChanKLine(
                    idx=prev.idx,
                    date=prev.date,
                    high=max(prev.high, curr.high),
                    low=max(prev.low, curr.low)  # 最高的低点
                )
            else:
                # 下降走势：取最低的高点和最低的低点
                merged = ChanKLine(
                    idx=prev.idx,
                    date=prev.date,
                    high=min(prev.high, curr.high),  # 最低的高点
                    low=min(prev.low, curr.low)
                )
            
            # 用合并后的K线替换前一根
            result[-1] = merged
            i += 1
        else:
            result.append(curr)
            i += 1
    
    # 递归处理：合并后可能产生新的包含关系
    if len(result) == len(klines):
        return result
    else:
        return merge_contain_klines(result, directions)
```

### 4.1 分型定义（PDF第17页）

```

│                      四种分型类型                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │

│                                                             │
│  4. 底分型：中间低                                           │
│     底分型 = 下降K线 + 底分型K线 + 上升K线                   │
│     high[i-1] >= high[i] <= high[i+1] 且                    │
│     low[i-1] > low[i] < low[i+1]                            │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  顶分型代表顶部形成，股价要下降                              │
│  底分型代表底部形成，股价要上升                              │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 分型识别代码实现

```python
from enum import Enum

class FractalType(Enum):
    """分型类型枚举"""
    TOP = "top"          # 顶分型
    BOTTOM = "bottom"    # 底分型
    UP = "up"            # 上升K线
    DOWN = "down"        # 下降K线

class Fractal:
    """分型数据结构"""
    def __init__(self, fractal_type, kline_idx, high, low, date):
        self.type = fractal_type
        self.kline_idx = kline_idx
        self.high = high
        self.low = low
        self.date = date

def identify_kline_direction(kline_prev, kline_curr):
    """
    识别单根K线的方向（上升K线或下降K线）
    
    规则：
    - 上升K线：出新新高、不出新低
    - 下降K线：出新低、不出新高
    """
    is_new_high = kline_curr.high > kline_prev.high
    is_new_low = kline_curr.low < kline_prev.low
    is_no_new_low = kline_curr.low >= kline_prev.low
    is_no_new_high = kline_curr.high <= kline_prev.high
    
    if is_new_high and is_no_new_low:
        return FractalType.UP
    elif is_new_low and is_no_new_high:
        return FractalType.DOWN
    else:
        # 非标准K线，返回None或根据上下文判断
        return None

def identify_fractals(klines):
    """
    识别所有分型（顶分型和底分型）
    
    顶分型定义：三根K线组成，中间一根最高
    底分型定义：三根K线组成，中间一根最低
    
    参数：
    - klines: 处理完包含关系后的缠论K线列表
    
    返回：
    - fractals: 分型列表
    - directions: 每根K线的方向列表
    """
    if len(klines) < 3:
        return [], []
    
    fractals = []
    directions = []
    
    for i in range(len(klines)):
        # 识别单根K线方向
        if i == 0:
            direction = FractalType.UP if klines[i].high > klines[i-1].high else FractalType.DOWN
        else:
            prev = klines[i-1]
            curr = klines[i]
            
            is_new_high = curr.high > prev.high
            is_new_low = curr.low < prev.low
            is_no_new_low = curr.low >= prev.low
            is_no_new_high = curr.high <= prev.high
            
            if is_new_high and is_no_new_low:
                direction = FractalType.UP
            elif is_new_low and is_no_new_high:
                direction = FractalType.DOWN
            else:
                # 包含关系已处理，这里理论上不会出现
                direction = directions[-1] if directions else FractalType.UP
        
        directions.append(direction)
        
        # 识别分型（需要三根K线）
        if 1 <= i < len(klines) - 1:
            prev = klines[i-1]
            curr = klines[i]
            next_k = klines[i+1]
            
            # 顶分型识别：中间高
            if (prev.high < curr.high > next_k.high and
                prev.low >= curr.low <= next_k.low):
                fractals.append(Fractal(
                    fractal_type=FractalType.TOP,
                    kline_idx=i,
                    high=curr.high,
                    low=curr.low,
                    date=curr.date
                ))
            
            # 底分型识别：中间低
            elif (prev.high >= curr.high <= next_k.high and
                  prev.low > curr.low < next_k.low):
                fractals.append(Fractal(
                    fractal_type=FractalType.BOTTOM,
                    kline_idx=i,
                    high=curr.high,
                    low=curr.low,
                    date=curr.date
                ))
    
    return fractals, directions
```

### 5.2 笔的数据结构

```python
class Stroke:
    """笔数据结构"""
    def __init__(self, stroke_id, direction, start_fractal, end_fractal, 
                 start_idx, end_idx, klines):
        self.id = stroke_id
        self.direction = direction      # 'up' 或 'down'
        self.start_fractal = start_fractal    # 起始分型
        self.end_fractal = end_fractal        # 结束分型
        self.start_idx = start_idx            # 起始K线索引
        self.end_idx = end_idx                # 结束K线索引
        self.high = max(start_fractal.high, end_fractal.high)
        self.low = min(start_fractal.low, end_fractal.low)
        self.klines = klines                  # 组成笔的所有K线
        self.verified = False                 # 是否已确认
    
    def __repr__(self):
        return f"Stroke({self.id}, {self.direction}, " \
               f"idx={self.start_idx}-{self.end_idx}, " \
               f"verified={self.verified})"
```

### 5.3 笔的构建算法

```python
def build_strokes(fractals, klines, min_klines=5):
    """
    构建笔
    
    参数：
    - fractals: 分型列表
    - klines: K线列表（处理完包含关系后）
    - min_klines: 最小K线数量（默认5根）
    
    返回：笔列表
    
    算法流程：
    1. 从第一个分型开始
    2. 寻找与之对应的相反分型
    3. 确认两者之间的K线数量 >= min_klines
    4. 满足条件则形成笔，否则继续等待
    5. 一笔结束后，下一笔自动开始
    """
    if len(fractals) < 2:
        return []
    
    strokes = []
    stroke_id = 0
    i = 0
    
    while i < len(fractals) - 1:
        current_fractal = fractals[i]
        next_fractal = fractals[i + 1]
        
        # 判断分型是否匹配
        is_match = False
        
        if current_fractal.type == FractalType.BOTTOM:
            # 底分型后应出现顶分型
            if next_fractal.type == FractalType.TOP:
                is_match = True
                direction = 'up'
                expected_next_type = FractalType.TOP
        
        elif current_fractal.type == FractalType.TOP:
            # 顶分型后应出现底分型
            if next_fractal.type == FractalType.BOTTOM:
                is_match = True
                direction = 'down'
                expected_next_type = FractalType.BOTTOM
        
        if is_match:
            # 计算两分型之间的K线数量
            start_idx = current_fractal.kline_idx
            end_idx = next_fractal.kline_idx
            kline_count = end_idx - start_idx + 1
            
            # 检查是否满足最小K线数量要求
            if kline_count >= min_klines:
                # 形成笔
                stroke = Stroke(
                    stroke_id=stroke_id,
                    direction=direction,
                    start_fractal=current_fractal,
                    end_fractal=next_fractal,
                    start_idx=start_idx,
                    end_idx=end_idx,
                    klines=klines[start_idx:end_idx+1]
                )
                strokes.append(stroke)
                stroke_id += 1
                i += 2  # 跳到下一个分型对
            else:
                # K线数量不足，继续等待下一个分型
                i += 1
        else:
            # 分型不匹配，跳到下一个
            i += 1
    
    # 笔的确认：新笔需要更新的笔出现后才能确认
    verify_strokes(strokes)
    
    return strokes

def verify_strokes(strokes):
    """
    确认笔
    
    规则：新笔虽然出现了，但还不能被确认
          新笔要等更新的笔出现后才能确立
    
    实现：最后一笔暂不确定，待下一笔出现后确认
    """
    for i, stroke in enumerate(strokes):
        if i < len(strokes) - 1:
            stroke.verified = True
        else:
            stroke.verified = False
```

### 5.5 笔的完整性验证

```python
def validate_stroke(stroke, min_klines=5):
    """
    验证笔的完整性
    
    必须满足以下条件：
    1. 方向正确
    2. 分型对应（顶底匹配）
    3. K线数量 >= 最小要求
    4. 中间K线不属于分型
    """
    errors = []
    
    # 1. 检查方向与分型匹配
    if stroke.direction == 'up':
        if stroke.start_fractal.type != FractalType.BOTTOM:
            errors.append("上升笔必须以底分型开始")
        if stroke.end_fractal.type != FractalType.TOP:
            errors.append("上升笔必须以顶分型结束")
    else:  # down
        if stroke.start_fractal.type != FractalType.TOP:
            errors.append("下降笔必须以顶分型开始")
        if stroke.end_fractal.type != FractalType.BOTTOM:
            errors.append("下降笔必须以底分型结束")
    
    # 2. 检查K线数量
    kline_count = stroke.end_idx - stroke.start_idx + 1
    if kline_count < min_klines:
        errors.append(f"K线数量不足：{kline_count} < {min_klines}")
    
    # 3. 检查分型索引
    if stroke.start_fractal.type == stroke.end_fractal.type:
        errors.append("顶底必须对应，不能出现同类型分型")
    
    return len(errors) == 0, errors
```

```python
class Segment:
```

### 6.3 线段构建算法

```python
def has_overlap(stroke1, stroke2):
    """
    判断两笔是否有重叠
    
    重叠定义：一笔的高点与另一笔的低点有交叉
    """
    # stroke1的高点 >= stroke2的低点 且 stroke1的低点 <= stroke2的高点
    return (stroke1.high >= stroke2.low) and (stroke1.low <= stroke2.high)

def check_overlap_in_strokes(strokes):
    """
    检查连续三笔之间是否有重叠
    
    线段要求三笔在价位上有重叠部分
    """
    if len(strokes) < 3:
        return False
    
    # 检查前三笔是否有重叠
    for i in range(len(strokes) - 2):
        s1 = strokes[i]
        s2 = strokes[i + 1]
        s3 = strokes[i + 2]
        
        # 检查s1与s2、s2与s3是否有重叠
        if has_overlap(s1, s2) and has_overlap(s2, s3):
            return True
    
    return False

def build_segments(strokes):
    """
    构建线段
    
    参数：
    - strokes: 笔列表
    
    返回：线段列表
    
    算法流程：
    1. 从第一笔开始
    2. 连续三笔有重叠则形成线段
    3. 等待反向线段出现后确认当前线段
    """
    if len(strokes) < 3:
        return []
    
    segments = []
    segment_id = 0
    i = 0
    
    while i <= len(strokes) - 3:
        # 检查前三笔是否有重叠
        candidate_strokes = strokes[i:i+3]
        
        if check_overlap_in_strokes(candidate_strokes):
            # 形成线段
            direction = candidate_strokes[0].direction
            
            segment = Segment(
                segment_id=segment_id,
                direction=direction,
                strokes=candidate_strokes,
                start_idx=i,
                end_idx=i + 2
            )
            segments.append(segment)
            segment_id += 1
            i += 3  # 移动到下一组
        else:
            i += 1  # 继续寻找
    
    # 确认线段
    verify_segments(segments)
    
    return segments

def verify_segments(segments):
    """
    确认线段
    
    规则：当前线段需要被反向线段确认
    """
    for i, segment in enumerate(segments):
        if i < len(segments) - 1:
            segment.verified = True
        else:
            segment.verified = False
```

```python
class CentralPivot:
    """中枢数据结构"""
    def __init__(self, pivot_id, direction, component_type,
                 start_idx, end_idx, components,ZG, ZD):
        self.id = pivot_id

        self.start_idx = start_idx          # 起始组件索引

        return self.ZG - self.ZD

        return self.ZD <= price <= self.ZG
```

### 7.3 中枢构建算法

```python
def find_overlap_range(strokes):
    """
    找重叠区域
    
    重叠区域 = 高点里的最低 - 低点里的最高
    
    示意图：
    ┌──────────────────────────────────────┐
    │                                      │
    │    ┌────┐                            │
    │    │    │ ← 高点1                    │
    │    └────┘                            │
    │         ┌────────┐                   │
    │         │        │ ← 高点2            │
    │         └────────┘                   │
    │                                      │
    │  ──────────                          │
    │      ↑                               │
    │   低点1                              │
    │              ──────────               │
    │                 ↑                    │
    │              低点2                    │
    │                                      │
    └──────────────────────────────────────┘
    
    ZG = min(高点1, 高点2)  # 高点里的最低
    ZD = max(低点1, 低点2)  # 低点里的最高
    """
    if len(strokes) < 3:
        return None, None
    
    # 收集所有高点和低点
    highs = [s.high for s in strokes]
    lows = [s.low for s in strokes]
    
    # 重叠区域
    ZG = min(highs)  # 高点里的最低
    ZD = max(lows)   # 低点里的最高
    
    return ZG, ZD

def determine_pivot_direction(strokes):
    """
    判断中枢方向
    
    规则：
    - 上升走势的中枢：下-上-下结构（高低高低）
    - 下降走势的中枢：上-下-上结构
    """
    if len(strokes) < 3:
        return None
    
    # 检查是否是下-上-下结构
    # 即：第一笔低 < 第二笔高 > 第三笔低
    if (strokes[0].low < strokes[1].high > strokes[2].low and
        strokes[0].high >= strokes[1].low <= strokes[2].high):
        return 'up'
    
    # 检查是否是上-下-上结构
    if (strokes[0].high > strokes[1].low < strokes[2].high and
        strokes[0].low <= strokes[1].high >= strokes[2].low):
        return 'down'
    
    return None

def build_pivots(strokes):
    """
    构建中枢
    
    参数：
    - strokes: 笔列表
    
    返回：中枢列表
    
    算法流程：
    1. 从第一笔开始，每次取连续5笔
    2. 判断是否形成中枢结构（下-上-下或上-下-上）
    3. 找到重叠区域（ZG、ZD）
    4. 等待走出段确认中枢完成
    """
    if len(strokes) < 5:
        return []
    
    pivots = []
    pivot_id = 0
    i = 0
    
    while i <= len(strokes) - 5:
        # 取连续5笔进行分析
        candidate = strokes[i:i+5]
        
        # 判断中枢方向
        direction = determine_pivot_direction(candidate)
        
        if direction:
            # 找到重叠区域
            # 使用前三笔来确定中枢本体（进入段+中枢+走出段的第一笔）
            ZG, ZD = find_overlap_range(candidate[1:4])  # 中枢本体
            
            if ZG is not None and ZD is not None and ZG > ZD:
                # 形成有效中枢
                pivot = CentralPivot(
                    pivot_id=pivot_id,
                    direction=direction,
                    component_type='stroke',
                    start_idx=i,
                    end_idx=i + 4,
                    components=candidate,
                    ZG=ZG,
                    ZD=ZD
                )
                pivots.append(pivot)
                pivot_id += 1
                i += 5  # 移动到下一组
            else:
                i += 1
        else:
            i += 1
    
    return pivots
```

```python
class TrendType(Enum):
    """走势类型"""
    CONSOLIDATION = "consolidation"  # 盘整
    TREND = "trend"                   # 趋势

class Trend:
    """走势数据结构"""
    def __init__(self, trend_id, trend_type, direction, pivots, parts):
        self.id = trend_id
        self.type = trend_type
        self.direction = direction
        self.pivots = pivots              # 中枢列表
        self.parts = parts                 # a, b, c, B, c...
        self.verified = False
    
    def __repr__(self):
        return f"Trend({self.id}, {self.type}, {self.direction}, " \
               f"pivots={len(self.pivots)})"

def identify_trend_type(pivots):
    """
    识别走势类型
    
    规则：
    - 1个中枢 → 盘整走势
    - 2个或以上同级别、同方向的中枢 → 趋势走势
    - 中枢之间不能有重合
    """
    if len(pivots) == 0:
        return TrendType.CONSOLIDATION
    
    if len(pivots) == 1:
        return TrendType.CONSOLIDATION
    
    # 检查是否有多个中枢
    for i in range(len(pivots) - 1):
        pivot1 = pivots[i]
        pivot2 = pivots[i + 1]
        
        # 检查方向是否一致
        if pivot1.direction != pivot2.direction:
            continue
        
        # 检查是否同级别（暂时简化处理：假设同级别）
        # 检查是否有重合
        if (pivot1.ZG >= pivot2.ZD and pivot1.ZD <= pivot2.ZG):
            # 有重合，中枢扩张
            continue
        
        # 找到符合条件的两个中枢，形成趋势走势
        return TrendType.TREND
    
    return TrendType.CONSOLIDATION
```

### 9.2 完整代码结构

```python
class ChanLunAnalyzer:
    """
    缠论分析器
    
    完整流程：
    1. 原始K线 → 缠论K线
    2. 缠论K线 → 包含关系处理
    3. 处理后K线 → 分型识别
    4. 分型 → 笔构建
    5. 笔 → 线段构建
    6. 线段 → 中枢提取
    7. 中枢 → 走势类型
    """
    
    def __init__(self, min_klines=5):
        self.min_klines = min_klines
        self.raw_klines = []
        self.chan_klines = []
        self.directions = []
        self.fractals = []
        self.strokes = []
        self.segments = []
        self.pivots = []
        self.trends = []
    
    def load_data(self, df):
        """
        加载数据
        
        参数：
        - df: DataFrame，包含 date, open, high, low, close 列
        """
        self.raw_klines = []
        for idx, row in df.iterrows():
            kline = ChanKLine(
                idx=idx,
                date=row['date'],
                high=float(row['high']),
                low=float(row['low'])
            )
            self.raw_klines.append(kline)
    
    def to_chan_kline(self):
        """转换为缠论K线（只保留最高点和最低点）"""
        self.chan_klines = self.raw_klines.copy()
    
    def process_contain(self):
        """处理包含关系"""
        # 首先计算初始方向
        self.directions = self._calculate_initial_directions()
        # 处理包含关系
        self.chan_klines = merge_contain_klines(self.chan_klines, self.directions)
    
    def identify_fractals(self):
        """识别分型"""
        self.fractals, self.directions = identify_fractals(self.chan_klines)
    
    def build_strokes(self):
        """构建笔"""
        self.strokes = build_strokes(self.fractals, self.chan_klines, self.min_klines)
    
    def build_segments(self):
        """构建线段"""
        self.segments = build_segments(self.strokes)
    
    def extract_pivots(self):
        """提取中枢"""
        self.pivots = build_pivots(self.strokes)
    
    def analyze_trends(self):
        """分析走势"""
        trend_type = identify_trend_type(self.pivots)
        self.trends.append(Trend(0, trend_type, 
                                 self.pivots[0].direction if self.pivots else None,
                                 self.pivots, {}))
    
    def analyze(self, df):
        """
        执行完整分析流程
        """
        self.load_data(df)
        self.to_chan_kline()
        self.process_contain()
        self.identify_fractals()
        self.build_strokes()
        self.build_segments()
        self.extract_pivots()
        self.analyze_trends()
        
        return {
            'klines': self.chan_klines,
            'directions': self.directions,
            'fractals': self.fractals,
            'strokes': self.strokes,
            'segments': self.segments,
            'pivots': self.pivots,
            'trends': self.trends
        }
```

```python
import matplotlib.pyplot as plt
import matplotlib.patches as patches

def visualize_chanlun(chan_data, output_path):
    """
    缠论可视化
    
    参数：
    - chan_data: analyze()返回的分析结果
    - output_path: 输出图片路径
    """
    fig, ax = plt.subplots(figsize=(20, 10))
    
    klines = chan_data['klines']
    fractals = chan_data['fractals']
    strokes = chan_data['strokes']
    segments = chan_data['segments']
    pivots = chan_data['pivots']
    
    # 1. 绘制K线
    for k in klines:
        ax.plot([k.idx, k.idx], [k.low, k.high], 'k-', linewidth=1)
    
    # 2. 标记分型
    for f in fractals:
        if f.type == FractalType.TOP:
            ax.scatter(f.kline_idx, f.high, marker='v', c='red', s=100)
        else:
            ax.scatter(f.kline_idx, f.low, marker='^', c='green', s=100)
    
    # 3. 绘制笔
    for s in strokes:
        x = [s.start_idx, s.end_idx]
        y = [s.start_fractal.high if s.direction == 'up' else s.start_fractal.low,
             s.end_fractal.high if s.direction == 'up' else s.end_fractal.low]
        color = 'green' if s.direction == 'up' else 'red'
        ax.plot(x, y, color=color, linewidth=2, label=f'Stroke {s.id}')
    
    # 4. 绘制中枢
    for p in pivots:
        rect = patches.Rectangle(
            (p.start_idx, p.ZD),
            p.end_idx - p.start_idx + 1,
            p.ZG - p.ZD,
            linewidth=2,
            edgecolor='blue',
            facecolor='lightblue',
            alpha=0.5
        )
        ax.add_patch(rect)
        ax.text(p.start_idx, p.ZG, f'ZG={p.ZG:.2f}', fontsize=8)
        ax.text(p.start_idx, p.ZD, f'ZD={p.ZD:.2f}', fontsize=8)
    
    ax.set_xlabel('K线序号')
    ax.set_ylabel('价格')
    ax.set_title('缠论分析图')
    ax.grid(True, alpha=0.3)
    
    plt.tight_layout()
    plt.savefig(output_path, dpi=150)
    plt.close()
```

## 附录：关键规则速查表

### A1. 缠论K线规则

| 规则 | 说明 |
|------|------|
| 只看高点和低点 | 忽略开盘价、收盘价 |
| 包含关系处理 | 先判断方向：上升取最高，下降取最低 |
| 处理顺序 | 从左到右依次处理 |

### A2. 分型规则

| 分型 | 定义 | 含义 |
|------|------|------|
| 上升K线 | 出新高、不出新低 | 多方占优 |
| 下降K线 | 出新低、不出新高 | 空方占优 |
| 顶分型 | 中间高，像"品"字 | 股价要下降 |
| 底分型 | 中间低 | 股价要上升 |

### A3. 笔规则

| 规则 | 说明 |
|------|------|
| 组成 | 顶分型 + 中间K线 + 底分型 |
| 方向 | 上升笔/下降笔 |
| 最小K线数 | 5根 |
| 连续性 | 一笔结束后必跟相反方向的一笔 |
| 确认 | 新笔需等更新的笔确认 |

### A4. 线段规则

| 规则 | 说明 |
|------|------|
| 组成 | 至少3笔 |
| 重叠要求 | 三笔价位有重叠 |
| 方向 | 向上线段/向下线段 |
| 终结 | 需反向线段确认 |

### A5. 中枢规则

| 规则 | 说明 |
|------|------|
| 最小结构 | 5根线段/笔 |
| 上升中枢 | 下-上-下结构 |
| 下降中枢 | 上-下-上结构 |
| 画法 | 找重叠区域（ZG、ZD） |

### A6. 走势规则

| 类型 | 定义 |
|------|------|
| 盘整走势 | 1个中枢 |
| 趋势走势 | 2个+同级别、同方向的中枢 |
| 中枢扩张 | 两中枢重合则变为一个中枢 |
