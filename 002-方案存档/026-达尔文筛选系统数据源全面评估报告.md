## 一、数据需求分析

根据达尔文筛选系统第一层的需求，需要以下类型的数据：

| 数据类型 | 用途 | 数据源优先级 |
|----------|------|--------------|
| **股票列表** | 全市场扫描 | 高 |
| **财务报表**（资产负债表、利润表、现金流量表）| 计算财务指标 | 高 |
| **实时行情**（市值、股价）| 排除ST股、计算市值 | 高 |
| **分红历史** | 计算连续分红年限 | 中 |
| **实时资金流向** | 备用 | 低 |

## 二、数据源全面评估

### 2.1 AKShare（东方财富源）

**状态**: ⚠️ 当前网络连接异常

**特点**:
- 数据源：东方财富、新浪财经等
- 无需注册
- 实时性强

**财务数据接口**:
- `stock_balance_sheet_by_report_em()` - 资产负债表
- `stock_profit_statement_by_report_em()` - 利润表
- `stock_cash_flow_sheet_by_report_em()` - 现金流量表

**问题**: 连接东方财富服务器被断开

**替代接口**:
- `stock_financial_analysis_indicator()` - 财务分析指标（可能更稳定）

---

### 2.2 Tushare Pro

**状态**: ✅ 推荐（需要注册）

**特点**:
- 数据全面（股票、基金、期货、宏观）
- 基础数据免费
- 注册地址：https://tushare.pro

**获取流程**:
1. 注册账号
2. 实名认证
3. 获取Token
4. 调用API

**财务数据接口**:
- `pro.balancesheet()` - 资产负债表
- `pro.income()` - 利润表
- `pro.cashflow()` - 现金流量表
- `pro.fina_indicator()` - 财务指标

**积分要求**:
- 基础积分：注册即送
- 财务数据：需要120积分
- 可通过完成任务提升积分

---

### 2.3 BaoStock

**状态**: ⚠️ 财务数据受限

**特点**:
- 专为A股设计
- 无需注册
- 分钟级K线数据完善

**财务数据接口**:
- `query_profit_data()` - 利润表
- `query_balance_data()` - 资产负债表

**问题**: 免费版财务数据返回空

**可用数据**:
- ✅ 股票列表
- ✅ K线数据
- ❌ 财务数据（受限）

---

### 2.4 eFinance

**状态**: 🆕 新发现，潜力较大

**特点**:
- 个人开发的免费开源库
- 基于东方财富数据
- 声称无调用限制

**安装**: `pip install efinance`

**可用功能**:
- 股票行情数据
- 财务数据（待验证）
- 基金、期货数据

---

### 2.5 东方财富直接API

**状态**: ✅ 可用（需要爬虫实现）

**特点**:
- 稳定、无调用限制
- 需要解析HTML

**接口地址**:

## 四、推荐实施方案

### 方案一：Tushare Pro（推荐）

**优点**:
- 数据最全面
- 接口稳定
- 有官方支持

**步骤**:
1. 访问 https://tushare.pro 注册
2. 完成实名认证
3. 在"个人主页"获取Token
4. 设置Token到环境变量或配置

**配置示例**:
```python
import os
os.environ['TUSHARE_TOKEN'] = 'your_token_here'
```

---

### 方案二：eFinance（备选）

**优点**:
- 无需注册
- 安装即用

**安装测试**:
```bash
pip install efinance
```

---

### 方案三：多数据源切换（最优）

**架构**:
1. 优先使用 AKShare（当前网络问题，暂不可用）
2. 备用 Tushare Pro（需注册）
3. 备选 eFinance（待验证）
4. 应急 BaoStock（仅K线可用）

**实现要点**:
- 数据源管理器自动切换
- 缓存机制防止重复请求
- 错误重试机制
