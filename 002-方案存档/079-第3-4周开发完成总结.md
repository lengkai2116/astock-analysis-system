## 四、API接口清单

### 4.1 因子管理

| 接口 | 方法 | 功能 |
|-----|------|------|
| `/api/factors` | GET | 获取所有因子 |
| `/api/factors/categories` | GET | 获取因子类别 |
| `/api/factors/sources` | GET | 获取因子来源 |
| `/api/factors/<name>` | GET | 获取因子详情 |
| `/api/factors/calculate/<name>` | POST | 计算单个因子 |
| `/api/factors/calculate-combination` | POST | 计算因子组合 |
| `/api/factors/evaluate` | POST | 评估因子质量 |
| `/api/factors/precompute` | POST | 预计算因子 |

### 4.2 组合管理

| 接口 | 方法 | 功能 |
|-----|------|------|
| `/api/factors/combinations` | GET | 获取所有组合 |
| `/api/factors/combinations/<id>` | GET | 获取组合详情 |
| `/api/factors/combinations` | POST | 创建组合 |
| `/api/factors/combinations/<id>` | PUT | 更新组合 |
| `/api/factors/combinations/<id>` | DELETE | 删除组合 |
| `/api/factors/combinations/default` | GET | 获取默认组合 |
| `/api/factors/combinations/<id>/backtest` | POST | 运行组合回测 |

### 4.3 策略流水线

| 接口 | 方法 | 功能 |
|-----|------|------|
| `/api/factors/pipeline/screen` | POST | 运行筛选流水线 |
| `/api/factors/pipeline/strategies` | GET | 列出可用策略 |
