# A股分析系统第四阶段 - AI研究流水线与报告中心

## 概述

第四阶段实现了完整的AI研究流水线与报告中心，包括AI策略生成、研究流水线编排、多格式报告生成等功能。

## 新增功能

### 1. AI策略生成服务

**文件**: `backend/app/services/ai_strategy_generator.py`

**功能**:
- `AIStrategyGenerator` 类 - AI策略生成器核心类
- `generate_indicator_from_description()` - 从自然语言描述生成量化指标
  - 支持检测指标类型（RSI、MACD、均线、布林带等）
  - 自动提取参数（周期、阈值等）
  - 生成计算公式和信号逻辑
  - 输出可执行的代码模板
- `interpret_backtest_result()` - AI解读回测结果
  - 计算综合评分
  - 分析优势与劣势
  - 生成改进建议
  - 评估风险等级
  - 提供交易建议

**LLM集成接口**:
- 预留LM Studio本地大模型接口
- 预留OpenAI API接口
- 支持mock模式用于测试

### 2. 研究流水线编排器

**文件**: `backend/app/services/research_pipeline.py`

**功能**:
- `ResearchPipeline` 类 - 研究流水线核心类
- 完整流程定义:
  1. **用户输入** - 接收和处理用户的研究请求
  2. **AI生成** - 使用AI生成量化策略代码
  3. **质量检查** - 检查代码质量和安全性
  4. **沙盒执行** - 在沙盒环境中执行策略代码
  5. **回测验证** - 使用历史数据进行回测
  6. **AI解释** - AI解读回测结果
  7. **报告生成** - 生成完整的研究报告

**状态管理**:
- `PipelineStatus` - 流水线状态（待处理、运行中、已完成、失败、已取消）
- `StepStatus` - 步骤状态（待处理、运行中、已完成、失败、跳过）
- `PipelineResult` - 流水线执行结果，包含所有步骤详情

### 3. 报告生成器

**文件**: `backend/app/services/report_generator.py`

**功能**:
- `ReportGenerator` 类 - 报告生成器核心类
- 支持的报告类型:
  - `generate_single_stock_report()` - 单股票策略报告
  - `generate_backtest_report()` - 回测报告
  - `generate_research_report()` - 完整研究报告

**多格式输出**:
- **Markdown** - 纯文本格式，适合文档和笔记
- **HTML** - 网页格式，带样式，适合在线查看
- **JSON** - 结构化数据格式，适合程序处理

**模板系统**:
- 使用Jinja2模板引擎
- 默认Markdown和HTML模板
- 支持自定义模板目录

### 4. 报告API路由

**文件**: `backend/app/routes/reports.py`

**API接口**:

#### 生成报告
- **POST** `/api/v2/reports/generate`
- 请求体:
  ```json
  {
    "report_type": "single_stock" | "backtest" | "research",
    "ts_code": "600519.SH",
    "description": "当RSI低于30时买入",
    "start_date": "20250101",
    "end_date": "20250501",
    "format": "markdown" | "html" | "json",
    "initial_capital": 100000,
    "parameters": {}
  }
  ```

#### 获取报告列表
- **GET** `/api/v2/reports`
- 查询参数:
  - `report_type` - 报告类型过滤
  - `ts_code` - 股票代码过滤
  - `limit` - 返回数量限制（默认20）
  - `offset` - 偏移量

#### 获取报告详情
- **GET** `/api/v2/reports/<report_id>`

#### 删除报告
- **DELETE** `/api/v2/reports/<report_id>`

#### 从描述生成策略
- **POST** `/api/v2/reports/strategy/generate`
- 请求体:
  ```json
  {
    "description": "当RSI低于30时买入",
    "context": {}
  }
  ```

#### AI解读回测结果
- **POST** `/api/v2/reports/backtest/interpret`
- 请求体:
  ```json
  {
    "backtest_result": {...},
    "strategy_description": "..."
  }
  ```

#### 获取支持的格式
- **GET** `/api/v2/reports/formats`

#### 健康检查
- **GET** `/api/v2/reports/health`

## 使用示例

### 1. 生成研究报告（完整流水线）

```python
import asyncio
from app.services.research_pipeline import create_research_pipeline

pipeline = create_research_pipeline({
    'llm': {
        'type': 'lm_studio',
        'endpoint': 'http://localhost:1234/v1',
        'model': 'local-model'
    },
    'continue_on_error': False
})

request = {
    'description': '当RSI低于30时买入，高于70时卖出',
    'ts_code': '600519.SH',
    'start_date': '20250101',
    'end_date': '20250501',
    'initial_capital': 100000
}

result = await pipeline.execute(request)

if result.success:
    print(f"流水线完成，总评分: {result.get_step('ai_interpretation').result['overall_score']}")
else:
    print(f"流水线失败: {result.error}")
```

### 2. 生成单股票策略报告

```python
from app.services.report_generator import create_report_generator

generator = create_report_generator()

report = generator.generate_single_stock_report(
    stock_data={
        'ts_code': '600519.SH',
        'name': '贵州茅台',
        'start_date': '20250101',
        'end_date': '20250501'
    },
    strategy_data={
        'indicator_type': 'rsi',
        'formula': 'RSI(close, 14)',
        'signal': '当RSI<30时买入，RSI>70时卖出'
    },
    format='html'
)

# 保存报告
generator.save_report(report, 'report.html')
```

### 3. AI解读回测结果

```python
from app.services.ai_strategy_generator import AIStrategyGenerator

generator = AIStrategyGenerator({
    'type': 'mock'  # 或 'lm_studio', 'openai'
})

interpretation = generator.interpret_backtest_result(
    backtest_result={
        'metrics': {
            'total_return': 0.25,
            'sharpe_ratio': 1.85,
            'max_drawdown': -0.08,
            'win_rate': 0.62
        },
        'trades': []
    },
    strategy_description='基于RSI的均值回归策略'
)

print(f"总体评分: {interpretation['overall_score']}")
print(f"风险等级: {interpretation['risk_level']}")
```

## 配置说明

### LLM配置

可以在环境变量中配置LLM:

```bash
# LM Studio配置
export LLM_TYPE=lm_studio
export LLM_ENDPOINT=http://localhost:1234/v1
export LLM_MODEL=local-model

# OpenAI配置
export LLM_TYPE=openai
export OPENAI_API_KEY=your-api-key
```

### 报告存储

报告默认存储在:
- 元数据: `reports/metadata/`
- 可通过配置修改存储路径

## 注意事项

1. **LLM服务**: AI功能需要LLM服务支持（LM Studio或OpenAI），如未配置将使用mock模式
2. **数据依赖**: 回测功能需要股票历史数据，确保Tushare或其他数据源可用
3. **异步执行**: 研究流水线使用异步执行，前端需要轮询或使用WebSocket获取进度
4. **模板定制**: 可在 `backend/templates/` 目录自定义报告模板
5. **安全检查**: 代码执行前会进行质量检查，检测潜在危险代码
