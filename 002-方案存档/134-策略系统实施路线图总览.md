### 1.1 三个关键定位

| 组件 | 定位 | 说明 |
|------|------|------|
| **策略系统** | 信号/建议生成器 | 输出 `signal + confidence + entry_zone + risk_line + target_zone + position_suggestion + holding_period + evidence + risk_notes`。不参与任何实盘操作 |
| **回测引擎** | 赢率证据生成器 | 对每类信号类型统计历史表现：胜率、盈亏比、夏普、条件概率等。输出作为 evidence 的一部分附在策略信号后。不是实盘模拟 |
| **账户管理** | 实盘记录+复盘 | 用户自行录入买卖记录，系统结合当时策略信号输出做信号匹配度检查，生成复盘报告。参见125号文档 |

### 1.2 回测与账户管理的关系

```
策略系统输出信号 → 用户自行决策是否执行
                        │
                  用户执行 → 在账户管理录入交易
                        │
                  复盘时 → 系统对比策略信号 vs 用户实际操作
                        │
                  输出信号匹配度、策略合规率、偏差分析
                        │
                  用于评估策略质量和优化方向
```

### 3.5 阶段4：回测赢率评估体系

**目标**: 改造回测引擎，使其输出**信号类型的赢率证据**，而非实盘模拟

**核心改造**:

```python
class SignalWinRateEvaluator:
    """信号类型赢率评估器
    
    不模拟实盘交易，只回答：历史发出这类信号后，后续走势如何？
    """
    
    def evaluate(self, historical_signals: List[Dict], 
                 price_data: pd.DataFrame) -> Dict:
        """
        对历史的每类信号，统计后续 N 个交易日的表现
        
        输出:
        {
            'signal_type': 'S_BUY',
            'samples': 127,           # 历史信号次数
            'win_rate_5d': 0.63,       # 5日后上涨概率
            'win_rate_10d': 0.58,      # 10日后上涨概率
            'win_rate_20d': 0.61,      # 20日后上涨概率
            'avg_return_5d': 0.023,    # 5日平均收益率
            'avg_return_20d': 0.047,   # 20日平均收益率
            'max_drawdown_avg': -0.05, # 期间平均最大回撤
            'sharpe_5d': 1.2,          # 5日夏普
            'win_rate_with_divergence': 0.71,    # 有背离确认时
            'win_rate_without_divergence': 0.55, # 无背离确认时
            'win_rate_market_good': 0.68,        # 大盘好时
            'win_rate_market_poor': 0.45,        # 大盘差时
        }
    """
```
