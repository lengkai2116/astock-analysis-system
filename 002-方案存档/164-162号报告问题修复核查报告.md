### P2 — 中优先级

#### 3️⃣ backtest ECharts 硬编码色值 → ⚠️ 设计如此，无需修复

**报告描述：** `views/backtest/index.vue:542-832` 使用 `#2a2a2a`/`#94a3b8`/`#1e293b` 等硬编码值。

**当前代码状态：**
```css
/* 实际格式为 CSS 变量 + fallback 值 */
color: var(--text-secondary, #94a3b8);
background: var(--bg-surface, #1e293b);
```

**说明：** ECharts 配置项不支持 CSS 变量（canvas 上下文），fallback 十六进制是标准做法。CSS 变量在运行时生效，fallback 仅在变量未定义时使用。这是**防御性编程**，符合前端最佳实践。
