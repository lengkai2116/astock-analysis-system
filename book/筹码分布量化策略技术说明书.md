## 1. 策略概述与理论基础

### 1.1 核心理论

筹码分布理论认为：价格走势是主力投资者和散户投资者共同作用的结果。筹码分布直观反映了各类投资者的持仓成本分布，筹码在不同价位之间的转移过程揭示了主力操盘的方向和阶段。

**三个核心命题**：

| 命题 | 量化含义 |
|------|----------|
| 筹码分布 = 持仓成本分布 | 可计算每个价格区间的持仓量分布 |
| 筹码转移 = 趋势方向 | 低位→高位转移 = 回升趋势；高位→低位转移 = 下跌趋势 |
| 筹码峰 = 支撑/压力位 | 筹码密集区是天然的技术支撑位或压力位 |

### 1.2 策略类型

本策略属于**量价 + 筹码分布**复合型中低频策略，适用于日线级别操作。

### 1.3 操盘周期模型

```
建仓期 ──→ 洗盘期 ──→ 拉升期 ──→ 出货期 ──→ 下跌期
筹码低位集中  筹码换手     筹码向上转移  筹码高位集中  筹码向下发散
```

## 2. 数据需求说明

### 2.1 必需要素

| 数据项 | 类型 | 频率 | 历史长度 | 说明 |
|--------|------|------|----------|------|
| 日K线 OHLCV | 基础 | 日 | ≥3年 | 开盘/最高/最低/收盘/成交量 |
| 筹码分布数据 | 核心 | 日 | ≥3年 | 每个交易日各价位的持仓量分布 |
| 大单/超大单流向 | 辅助 | 日 | ≥3年 | 大单净流入/流出金额 |
| 换手率 | 辅助 | 日 | ≥3年 | 日换手率百分比 |
| 分时数据（可选） | 增强 | 分钟 | ≥1年 | 用于盘中信号确认 |

### 2.2 筹码分布数据格式

每个交易日返回一个数组，每个元素包含：

```
{
  "price": 价格水平,
  "volume_pct": 该价位的持仓量占比（百分比）,
  "profit_pct": 该价位持仓的盈亏比例
}
```

**关键衍生字段**：

| 衍生量 | 计算方式 |
|--------|----------|
| 筹码获利率 | `SUM(volume_pct) WHERE price < current_close` |
| 筹码亏损率 | `SUM(volume_pct) WHERE price > current_close` |
| 筹码集中度 | 覆盖80%持仓量的价格区间宽度 / 当前价格 × 100% |
| 筹码峰位置 | `argmax(volume_pct)` 对应的价格 |

### 2.3 大单流向数据

| 字段 | 说明 |
|------|------|
| `super_large_net_inflow` | 超大单净流入（亿元） |
| `super_large_net_outflow` | 超大单净流出（亿元） |
| `large_net_inflow` | 大单净流入（亿元） |
| `accum_net_inflow_20d` | 20日累计超大单净流入 |

## 3. 核心指标模块

### 3.1 ASR（浮筹指标）

**定义**：当前价格上下各5%范围内所覆盖的筹码数量占总筹码的百分比。

```
ASR = 价格区间[close × 0.95, close × 1.05]内的筹码量 / 总筹码量 × 100%
```

**量化阈值**：

| ASR 数值 | 状态 | 信号含义 |
|----------|------|----------|
| ASR < 20 | 低浮筹 | 价格远离密集成交区，趋势较强 |
| 20 ≤ ASR < 50 | 中等浮筹 | 正常状态 |
| 50 ≤ ASR < 80 | 高浮筹 | 筹码趋于集中，调整蓄势 |
| 80 ≤ ASR < 90 | 极高浮筹 | 密集成交区，突破前兆 |
| ASR ≥ 90 | 极限浮筹 | 极度集中，突破一触即发 |

**信号规则**：

```
买入预警：ASR 从 < 80 上升到 ≥ 80，且持续 ≥ 5 个交易日
突破确认：ASR 从 ≥ 80 回落到 < 50，且当日涨幅 > 2%
```

### 3.2 CYQKL（筹码穿透力指标）

**定义**：当日K线实体（开盘到收盘）穿透的筹码数量占总筹码的百分比。

```
CYQKL = K线实体覆盖价格区间内的筹码量 / 总筹码量 × 100%
```

注：阳线实体 = close - open；阴线实体 = open - close。

**量化阈值**：

| CYQKL 数值 | 穿透力度 | 信号含义 |
|-----------|----------|----------|
| CYQKL < 10 | 弱 | 正常波动，无操盘意义 |
| 10 ≤ CYQKL < 30 | 中等 | 主力异动迹象，关注 |
| 30 ≤ CYQKL < 50 | 强 | 主力明确介入 |
| 50 ≤ CYQKL < 60 | 很强 | 强力突破 |
| CYQKL ≥ 60 | 极强 | 极限突破，控盘力度最大 |

**增强规则**：当日 CYQKL 超过前20日平均 CYQKL 的 **2倍以上**，即使绝对值未达30，也视为主力异动信号。

### 3.3 SSRP（持仓成本价）

**定义**：全体投资者的加权平均持仓成本。

```
SSRP = Σ(price_i × volume_pct_i) / Σ(volume_pct_i)
```

**信号规则**：

```
买入：价格从下方突破 SSRP，且当日涨幅 > 2%、成交量 > 100日均量的 1.5 倍
卖出：价格从上方跌破 SSRP，且连续 2 个交易日收盘低于 SSRP
```

### 3.4 RSI（相对强弱指标）

**标准参数**：14日周期。

**量化阈值**：

| RSI 数值 | 状态 | 信号含义 |
|----------|------|----------|
| RSI < 20 | 极度超卖 | 底部反弹概率极高 |
| 20 ≤ RSI < 30 | 超卖 | 底部区域 |
| 30 ≤ RSI < 50 | 偏弱 | 调整中 |
| 50 ≤ RSI < 70 | 偏强 | 正常上升 |
| 70 ≤ RSI < 80 | 偏强/注意 | 接近超买 |
| RSI ≥ 80 | 超买 | 高位风险 |

**背离检测**：

```
顶背离（卖出信号）：价格创 20 日新高 且 RSI 未创 20 日新高
底背离（买入信号）：价格创 20 日新低 且 RSI 未创 20 日新低
```

**回调到位判定**：

```
RSI 从超买区（≥ 80）回落至 50 以下且企稳 ≥ 3 日 → 洗盘可能结束
RSI 回落至前期低点 ± 5 以内并反弹 → 调整结束确认
```

### 3.5 成交量指标

**100日等量线** = 过去100个交易日的日均成交量。

| 对比关系 | 数值条件 | 状态 |
|----------|----------|------|
| 放量 | 当日量 ≥ 100日均量 × 1.5 | 有效放量 |
| 显著放量 | 当日量 ≥ 100日均量 × 2.0 | 主力介入概率高 |
| 天量 | 当日量 ≥ 100日均量 × 3.0 | 主力强控盘 |
| 缩量 | 当日量 ≤ 100日均量 × 0.7 | 交投清淡 |
| 地量 | 当日量 ≤ 100日均量 × 0.3 | 底部信号 |

**量能趋势**：

```
量能持续放大：连续 5 日成交量 > 100日均量 → 主力持续介入
量能萎缩触底：连续 3 日成交量 < 100日均量 × 0.5 → 洗盘可能结束
```

### 3.6 换手率指标

| 换手率 | 状态 | 含义 |
|--------|------|------|
| < 2% | 低活跃度 | 冷门股，不适合操作 |
| 2% ~ 5% | 正常 | 一般关注 |
| 5% ~ 10% | 较高 | 活跃股，筹码转移快 |
| ≥ 10% | 极高 | 加速转移，高位警惕 |

### 3.7 筹码集中度指标

```
集中度 = (P95价格 - P5价格) / 当前价格 × 100%
其中 P95 为覆盖95%筹码的最高价，P5 为覆盖5%筹码的最低价
```

| 集中度 | 状态 |
|--------|------|
| < 10% | 高度集中 |
| 10% ~ 20% | 较集中 |
| 20% ~ 40% | 分散 |
| > 40% | 高度发散 |

## 4. 形态识别模块

### 4.1 筹码峰类型识别

#### 4.1.1 单一筹码主峰（单峰密集）

**量化条件**：
1. 存在一个价格区间 P_peak，其筹码量占比 ≥ 30%
2. P_peak 的筹码量 ≥ 次高峰筹码量 × 2
3. 筹码集中度 < 15%

**信号含义**：主力持仓高度集中，突破在即。

#### 4.1.2 筹码双峰

**量化条件**：
1. 存在两个价格区间 P_high、P_low，各自筹码量占比 ≥ 15%
2. |P_high - P_low| / close ≥ 10%（两峰间距足够）
3. 两峰之间区域筹码量占比 < 20%

**信号含义**：
- 低位峰：主力持仓成本区
- 高位峰：散户持仓成本区

#### 4.1.3 筹码发散（多峰分散）

**量化条件**：
1. 不存在任何单一价格区间筹码量占比 ≥ 20%
2. 筹码集中度 > 30%
3. 筹码覆盖价格区间 / close > 50%

#### 4.1.4 脉冲筹码峰

**量化条件**：
1. 存在 3 个及以上筹码峰
2. 相邻峰间距 > 5%
3. 各峰筹码量占比在 5%~15% 之间

### 4.2 筹码转移检测

#### 4.2.1 筹码向上转移（拉升阶段）

**量化判定**：

```
条件1：低位筹码峰（底部 20% 价格区间内）筹码量占比 较 20 日前减少 ≥ 5%
条件2：高位筹码峰（顶部 20% 价格区间内）筹码量占比 较 20 日前增加 ≥ 5%
同时满足 → 筹码向上转移确认
```

#### 4.2.2 筹码向下转移（出货/下跌阶段）

**量化判定**：

```
条件1：高位筹码峰筹码量占比 较 20 日前减少 ≥ 5%
条件2：低位筹码峰筹码量占比 较 20 日前增加 ≥ 5%
同时满足 → 筹码向下转移确认
```

#### 4.2.3 筹码转移速度

```
转移速度 = |Δ低位筹码占比| / 20日  （日化转移率）
速度快：转移速度 > 1.0%/日 → 行情加速
速度慢：转移速度 < 0.3%/日 → 蓄势阶段
```

## 5. 操盘阶段判断模块

### 5.1 阶段判定状态机

```
                ┌──────────────┐
                │   建仓阶段    │
                └──────┬───────┘
                       │ 低位筹码峰形成 + ASR→80以上
                       ▼
                ┌──────────────┐
                │   洗盘阶段    │
                └──────┬───────┘
                       │ 量能萎缩至地量 + RSI触底 + 价格未破低位峰
                       ▼
                ┌──────────────┐
                │   拉升阶段    │
                └──────┬───────┘
                       │ 低位筹码向高位持续转移 + 放量
                       ▼
                ┌──────────────┐
                │   出货阶段    │
                └──────┬───────┘
                       │ 高位筹码主峰形成 + 量能萎缩 + RSI背离
                       ▼
                ┌──────────────┐
                │   下跌阶段    │
                └──────────────┘
```

### 5.2 建仓阶段判定

**必要条件（同时满足 ≥ 3 条）**：

| 编号 | 条件 | 量化参数 |
|------|------|----------|
| J1 | 低位筹码峰形成 | 底部 30% 价格区间内筹码量 ≥ 40% |
| J2 | ASR 持续高位 | ASR ≥ 70 持续 ≥ 10 个交易日 |
| J3 | 脉冲大单频繁 | 过去 60 日内 ≥ 5 日出现超大单净流入 > 1 亿元 |
| J4 | 横盘时间充足 | 价格在 ±15% 区间内波动 ≥ 60 个交易日 |
| J5 | 换手率正常偏高 | 期间平均换手率在 3%~8% |
| J6 | 股价低位 | 价格处于 250 日价格区间下 30% 范围内 |

**确认信号**：J1 + (J2|J3|J4 中任意两条) → 建仓阶段确认。

### 5.3 洗盘阶段判定

**必要条件**：

| 编号 | 条件 | 量化参数 |
|------|------|----------|
| X1 | 股价回调 | 从阶段高点回落 10%~33% |
| X2 | 量能萎缩 | 成交量从放量降至接近 100 日均量 × 0.5 |
| X3 | RSI 回调 | RSI 从 ≥ 70 回落至 30~55 区间 |
| X4 | 筹码换手 | 高位筹码峰规模较洗盘前 ≥ 增加 10% |
| X5 | 价格未破主力成本 | 收盘价 > 低位筹码峰下限 |
| X6 | 横盘消化 | 回调后横盘 ≥ 20 个交易日 |

**确认信号**：X2 + X3 + X5（必选）+ (X1|X4|X6 中任意一条) → 洗盘阶段确认。

**洗盘深度参考**：

| 主力手法 | 回调幅度 | 持续时间 |
|----------|----------|----------|
| 缩量下挫打压 | 15%~33% | 10~20 交易日 |
| 冲高回落震荡 | 10%~25% | 5~15 交易日 |
| 横盘洗盘 | 5%~15% | 30~90 交易日 |
| 低开杀跌洗盘 | 15%~25% | 3~10 交易日 |

### 5.4 拉升阶段判定

**必要条件**：

| 编号 | 条件 | 量化参数 |
|------|------|----------|
| L1 | 低位筹码持续减少 | 底部 30% 区间筹码量 20 日内减少 ≥ 8% |
| L2 | 高位筹码持续增加 | 顶部 30% 区间筹码量 20 日内增加 ≥ 8% |
| L3 | 成交量放大 | 连续 5 日成交量 > 100 日均量 |
| L4 | 价格突破 | 收盘价 > 前 60 日最高收盘价 |
| L5 | ASR 快速回落 | ASR 从 ≥ 70 回落至 < 40 |
| L6 | CYQKL 增强 | CYQKL 较前 20 日均值 ≥ 增加 50% |

**确认信号**：L3 + L4 + (L1|L2 任一) → 拉升阶段确认。

### 5.5 出货阶段判定

**必要条件**：

| 编号 | 条件 | 量化参数 |
|------|------|----------|
| C1 | 高位筹码主峰形成 | 顶部 20% 区间筹码量 ≥ 50% |
| C2 | 低位筹码基本消失 | 底部 30% 区间筹码量 < 15% |
| C3 | RSI 顶背离 | 价格创 60 日新高 且 RSI 连续 2 次未创新高 |
| C4 | 量能萎缩 | 连续 5 日成交量 < 100 日均量 |
| C5 | 价格跌破高位筹码峰 | 收盘价 < 高位筹码峰下限 |
| C6 | 筹码获利率下降 | 筹码获利率从 > 80% 降至 < 35% |

**确认信号**：C1 + (C3|C4|C5 中任意两条) → 出货阶段确认。

### 5.6 下跌阶段判定

| 编号 | 条件 | 量化参数 |
|------|------|----------|
| D1 | 高位筹码峰被跌破 | 收盘价 < 最大筹码峰价格 × 0.98 |
| D2 | 筹码向下转移 | 高位筹码占比 20 日内减少 ≥ 5% |
| D3 | RSI < 50 且持续 | RSI 连续 10 日 < 50 |
| D4 | 换手率低迷 | 连续 10 日换手率 < 3% |

## 6. 信号生成逻辑

### 6.1 买入信号

#### 6.1.1 主买入信号（S_BUY）

```
触发条件（同时满足）：
  1. 操盘阶段 = 拉升阶段（刚确认 ≤ 5 个交易日）
  2. 价格突破 60 日最高价
  3. 成交量 ≥ 100 日均量 × 1.5
  4. CYQKL ≥ 20
  5. 筹码获利率 ≥ 60%
  
仓位：标准仓位的 70%
```

#### 6.1.2 洗盘结束买入信号（S_WASH_END）

```
触发条件（同时满足）：
  1. 操盘阶段 = 洗盘阶段
  2. 成交量降至地量（≤ 100 日均量 × 0.4）
  3. RSI 在 30~55 区间企稳 ≥ 3 日
  4. 收盘价 ≥ 低位筹码峰下限（主力成本区）
  5. 当日涨幅 > 0% 且成交量较前日放大 ≥ 20%
  
仓位：标准仓位的 50%
```

#### 6.1.3 超跌反弹买入信号（S_BOUNCE）

```
触发条件（同时满足）：
  1. 操盘阶段 = 下跌阶段
  2. 价格跌至历史低位筹码峰 ± 5% 范围内
  3. RSI 底背离确认
  4. 筹码获利率 < 20%
  
仓位：标准仓位的 30%
止损：跌破低位筹码峰下限 3%
```

### 6.2 卖出信号

#### 6.2.1 主卖出信号（S_SELL）

```
触发条件（满足任一）：
  A. 出货阶段确认：
     1. 高位筹码主峰形成（顶部 20% 区间筹码 ≥ 50%）
     2. RSI 顶背离（价格新高 + RSI 未创新高）
     3. 成交量萎缩至 100 日均量以下
     → 卖出 100%
  
  B. 价格跌破筹码主峰：
     1. 收盘价 < 最大筹码峰价格
     2. 筹码获利率骤降至 < 40%
     → 卖出 100%
  
  C. 高位冲高回落：
     1. 价格阶段涨幅 ≥ 80% 后单日冲高回落振幅 ≥ 7%
     2. 成交量 ≥ 100 日均量 × 2 但收阴
     → 卖出 50%
```

#### 6.2.2 洗盘止损信号（S_WASH_STOP）

```
触发条件：
  1. 价格跌破低位筹码峰下限（主力成本被破）
  2. 或价格跌幅超过从阶段高点计算 35%
  → 卖出全部仓位，止损出局
```

#### 6.2.3 高位背离卖出（S_DIVERG_SELL）

```
触发条件：
  RSI 与价格连续 2 个阶段顶背离：
    第一次背离：价格创新高，RSI 不创新高 → 减仓 30%
    第二次背离：价格再创新高，RSI 继续不创新高 → 减仓 70%
```

### 6.3 信号优先级

| 优先级 | 信号 | 处理方式 |
|--------|------|----------|
| 1（最高） | S_SELL（出货确认） | 立即全仓卖出 |
| 2 | S_WASH_STOP（止损） | 立即全仓卖出 |
| 3 | S_DIVERG_SELL | 按比例减仓 |
| 4 | S_BOUNCE（反弹买入） | 轻仓试探 |
| 5 | S_WASH_END（洗盘结束） | 半仓介入 |
| 6 | S_BUY（主买入） | 标准仓位介入 |

## 7. 风险控制与仓位管理

### 7.1 仓位分配框架

| 操盘阶段 | 最大仓位 | 单次建仓比例 | 加仓条件 |
|----------|----------|-------------|----------|
| 建仓阶段 | 30% | 10% | 建仓确认信号加 20% |
| 洗盘阶段 | 50% | 20% | 洗盘结束信号加 30% |
| 拉升阶段 | 70% | 40% | 回调支撑位加 30% |
| 出货阶段 | 0% | - | 仅减仓不建仓 |
| 下跌阶段 | 30% | 15%（仅反弹） | RSI 背离加 15% |

### 7.2 止损规则

| 止损类型 | 触发条件 | 操作 |
|----------|----------|------|
| 硬止损 | 价格跌幅 > 入场价的 8% | 立即全仓卖出 |
| 跟踪止损 | 从最高价回撤 > 15% | 卖出 50% |
| 筹码止损 | 价格跌破入场时低位筹码峰下限 | 立即全仓卖出 |
| 时间止损 | 建仓后 20 个交易日未盈利 | 卖出 50% |

### 7.3 止盈规则

| 止盈类型 | 触发条件 | 操作 |
|----------|----------|------|
| 筹码止盈 | 低位筹码峰消失（底部 30% 区间筹码 < 10%） | 卖出 70% |
| RSI 止盈 | RSI ≥ 85 且连续 3 日 | 卖出 50% |
| 放量滞涨止盈 | 成交量 > 100 日均量 × 2 但收盘涨跌幅 < 0.5% | 卖出 30% |
| 移动止盈 | 从阶段高点回撤 10% | 卖出剩余全部 |

### 7.4 大盘环境过滤

```
大盘环境不佳时（沪深300 在 60 日均线下方）：
  - 总仓位上限降至 40%
  - 仅执行 S_BUY 和 S_BOUNCE
  - 降低仓位至标准的 50%

大盘环境良好时（沪深300 在 60 日均线上方）：
  - 总仓位上限 70%
  - 全信号执行
```

## 9. 实施难点与解决方案

### 9.1 难点一：筹码分布数据的获取

**问题**：A 股市场不直接提供筹码分布数据。

**解决方案**：

| 方案 | 可行性 | 说明 |
|------|--------|------|
| 数据服务商购买 | 高 | Wind/Choice/同花顺等提供筹码分布数据 |
| 公开API | 中 | 东方财富/同花顺有部分免费接口 |
| 自行估算 | 中 | 基于逐笔成交累计+衰减模型估算 |

**自行估算模型（备选方案）**：

```
基础假设：
  1. 每个交易日的成交量分布在该日价格区间内
  2. 历史持仓量按换手率衰减（衰减因子 = 日换手率）
  
简化公式：
  Chip[t][p] = Chip[t-1][p] × (1 - turnover_rate[t]) + Volume[t] × distribution(p, OHLC[t])
  
其中 distribution(p, OHLC[t]) 基于当日价格分布（假设均匀或正态分布）
```

### 9.2 难点二：主力vs散户筹码区分

**问题**：筹码分布不区分持有者身份。

**解决方案**：

1. **基于持仓稳定性区分**：连续 60 日以上未转移的筹码标记为「稳定持仓」（近似主力）
2. **基于价格位置区分**：底部 30% 区间的筹码默认为主力概率高
3. **基于大单流向辅助**：超大单净流入日的筹码增量标记为主力仓位

### 9.3 难点三：假突破与假跌破

**问题**：筹码峰突破/跌破可能是主力故意制造的假信号。

**解决方案**：

```
突破有效性过滤器：
  1. 突破日成交量 ≥ 100 日均量 × 1.5
  2. 突破后 3 日内未回落到突破前价格区间
  3. CYQKL ≥ 20 确认穿透深度

跌破有效性过滤器：
  1. 跌破日成交量 ≥ 100 日均量 × 0.8（非无量下跌）
  2. 跌破后 3 日内未收复筹码峰下限
  3. 筹码获利率 < 40% 持续 ≥ 3 日
```

### 9.4 难点四：不同市值的适用性差异

| 市值区间 | 筹码理论适用性 | 调整建议 |
|----------|----------------|----------|
| < 50 亿（小盘） | 高 | 标准参数 |
| 50~200 亿（中盘） | 较高 | 放宽成交量倍数 0.2 |
| 200~1000 亿（大盘） | 中等 | CYQKL 阈值降低 5，放宽 |
| > 1000 亿（超大） | 较低 | 筹码集中度阈值降低，不作为主策略 |

### 9.5 难点五：股灾/系统性风险

**解决方案**：

```
熔断规则：
  1. 大盘单日跌幅 > 5% → 所有仓位平仓 50%
  2. 大盘连续 3 日下跌且累计 > 8% → 清空所有仓位
  3. 个股连续跌停无法卖出 → 次日集合竞价市价卖出
```

### 9.6 难点六：新股的筹码数据不完整

**解决方案**：

- 上市不满 1 年的股票排除出选股池
- 或仅当可获取 ≥ 250 个交易日筹码数据时才纳入
