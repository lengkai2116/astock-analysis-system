# A 股研究 — 股票操作知识库操作手册

## 项目概览

这是一个基于 **LLM Wiki** 构建的股票操作知识库，通过 AI 自动解读多本股市电子书，生成结构化的知识体系。知识库运行在本地 LLM Wiki 桌面应用上，由 DeepSeek API 驱动。

## 知识库位置

- **LLM Wiki 项目名**: A股研究
- **项目路径**: `/Users/kalence/Desktop/未命名文件夹/A股研究`
- **API 地址**: `http://127.0.0.1:19828`
- **API Token**: 已设置环境变量 `LLM_WIKI_API_TOKEN`

## 知识库当前内容

知识库已从多本电子书中提取了以下知识模块（共 50+ 页面）：
- **看盘基础**: K线形态、均线系统、分时图战法、A股交易机制

## 核心工作流

### 查询知识库

当用户问"我的知识库里关于 X 的内容"时：

1. 搜索知识库：`POST /api/v1/projects/current/search` + 股票知识相关关键词
2. 读取匹配页面内容（使用 `includeContent: true` 减少往返）
3. 引用页面路径形成回答

### 导入新电子书

1. 将电子书放入 `/Users/kalence/Desktop/未命名文件夹/A股研究/raw/sources/`
2. 触发重新扫描：`POST /api/v1/projects/current/sources/rescan`
3. 告知用户在 LLM Wiki 桌面应用的活动面板查看进度

### 知识库维护

- **添加新源文件** → rescan 即可自动摄取
- **查询知识缺口** → 使用 graph API 查看孤立节点
- **跨概念关联** → 使用 search API 做语义搜索

## 注意事项

- API 默认使用 `current` 项目（即 A 股研究），多项目管理时才需切换 project ID
- 搜索 mode 为 keyword（向量搜索尚未启用），评分一般在 50-300+ 范围
- 文件内容读取上限 2MB，文字类型仅支持 md/txt/json/yaml/csv 等
- 如果 API 不可达，先确认 LLM Wiki 桌面应用是否在运行
